9. perbedaan char dan varchar


jadiin jawaban terbaik dong

Perbedaan antara Varchar dan Char terletak pada kecepatan access data dan penghematan space pada system,
•Varchar
Jika panjang character suatu data hanya 10 character maka system akan membaca panjang character tersebut 50 character
•Char
Jika penjang character suatu data itu hanya 1 character maka system akan tetap membacanya 1
Dari penjelasan diatas dapat disimpulkan bahwa Varchar akan di access lebih cepat dari pada char,namun type data char bisa menghemat space karena panjang character data 1 tetap dibaca 1 character sedangkan varchar panjang character 10 tetap dibaca 50.

int sum = 0;
