9. Bagaimana kenampakan takik lereng (break of slope) tersebut dapat terjadi pada tubuh Gunungapi Merapi strato tersebut?


Pada gunung api strato memiliki potensi besar pada lereng gunung api, pola persebaranya mengelilingi badan pada gunung api yang nantinya akan membentuk pola seperti sabuk yang mempunyai hubungan dengan perubahan struktur batuan pembentuknya yang ada di lereng gunung.

Pembahasan :

Saluran fluida panas yang dapat berupa wujud cair maupun panas atau lava ada di dalam bumi yang menuju ke permukaan bumi disebut dengan gunung api. Gunung api dapat terbentuk akibat dari kerak buimi yang mengalami pecahan sehingga terdapat patahan yang akan mengelurkan lava panas dari dalam bumi ke permukaan bumi.

Jenis gunung api

Gunung api mati, dimana gunung api mati telah berhenti untuk melakukan erupsi dan tidak akan pernah kembali meletus.Gunung api itirahat, dimana gunung api masih mempunyai aktivitas vulkaniknya tetapi masih dalam keadaan istirahat seperti gunung kelud yang sewaktu - waktu dapat meletus. Gunung strato atau kerucut, gunung yang dihasilkan akibat dari arupsi efusif dan eksplosit seperti pada gunung merapi yang ada di Yogyakarta.Gunung perisai, dimana dapur magma yang ada didalam gunung mempunyai sfat cair seperti gunung Mouna di Amerika Serikat.Gunung maar, terjadinya akibat dari pengaruh eksplosif yang dangkal seperti pada gunung Lamongan.
