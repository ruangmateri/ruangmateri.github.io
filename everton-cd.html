7. Apa perpedaan CD, CD-R, CD-RW?


1. CD-ROM  merupakan akronim dari “compact disc read-only memory” adalah sebuah piringan kompak dari jenis piringan optik (optical disc) yang dapat menyimpan data. Ukuran data yang dapat disimpan saat ini bisa mencapai  700MB atau 700 juta bita CD-ROM drive hanya bisa digunakan untuk membaca sebuah CD saja. Secara gari besar CD-ROM dibedakan menjadi 2 menurut tipenya yaitu : ATA/IDE  dan SCSI. Yang paling mendasari dari perbedaan tersebut adalah kecepatannya. Kalau ATA memiliki kecepatan 100-133Mbps sedangkan SCSI memiliki kecepatan kira-kira 150 Mbps. Untuk tipe SCSI biasanya ditemuka pada CR RW drive. Pada CD ROM terdapat tulisan 56X artinya kemampuan memberikan kecepatan transfer data sebesar 56 x150 Kbps. Tipe CD RW juga biasanya  dibedakan berdasarkan kemapuan membakar dan membaca. CD RW tipe 12x8x32 artinya memiliki kemampuan membakar pada CD R seccepat 12x, membakar pada CD RW secepat 8x, dan membaca CD R/CD RW/dengan kecepatan maksimal 32x.
2. CD-R adalah singkatan dari Compact Disk Recordable adalah piringan pipih berbahan polikarbonat berdiameter 120 mm. Untuk mengisi CD-R dengan data diperlukan sebuah perangkat drive khusus. Proses perekamnya juga dilakukan dengan perangkat lunak khusus untuk mem burning data ke dalam CD-R. Data yang ditulis didalamnya bersifat WORM ( Write Once Read Many ) yang menunjukan bahwa data hanya dapat diisi sekali dan selanjutnya hanya bisa dilihat / dibaca.
3. CD-RW merupakan pengembangan dari CD-R, sebenarnya dari bentuk sama dengan CD-R hanya sedikit berbeda dalam kegunaan dalam hal penyimpanan data. Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu.

CD = sebuah disc yang dapat menyimpan memory data
CD - R = sebuah disc yang hanya dapat menyimpan memory data sekali , setelah itu hanya dapat dibaca saja
CD-RW = sebuah disc yang dapat menyimpan data beberapa kali dan dapat dirubah kembali isi nya

CD = tergantung dari jenis dan type nya
CD - R = biasanya berwarna bening
CD - RW = biasanya berwarna ungu gelap

CD = tergantungf dari jenis nya
CD - R = dapat menampung sebuah memory sebesar ~ 700 Mb
CD - RW = dapat menampung sebua memory sebesar ~ 4 GB

Dapat diartikan jika kita telah burning data ke dalam CD-R, maka data tersebut sudah tidak bisa kita hapus lagi. Bisa dikatakan CD-R hanya dapat digunakan untuk satu kali pemakaian karena kita tidak bisa mengahapus ini data yang ada di dalamnya.

Kedua yaitu CD-RW yang merupakan pengembangan dari CD-R, sebenarnya dari bentuk sama dengan CD-R hanya sedikit berbeda dalam kegunaan dalam hal penyimpanan data. Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu.

Jadi seandainya  data yang ada pada CD-RW sudah tidak digunakan lagi, kita dapat menghapus dan menggantinya dengan yang baru.CD R tidak dapat mengopi file / video / gambar ke perangkat komputer atau laptop anda , dan harus melalui proses pembakaran dulu jika ingin menyalin video atau gambar tersebut pakai nero smart project, dan data yg telah disalin tidak dapat dihapus lagi, pemakaian hanya sekali saja.

31. Pengertian cd rom, cd-r,cd+r,cd rw


CD-ROM  merupakan akronim dari “compact disc read-only memory” adalah sebuah piringan kompak dari jenis piringan optik (optical disc) yang dapat menyimpan data. Ukuran data yang dapat disimpan saat ini bisa mencapai  700MB atau 700 juta bita CD-ROM drive hanya bisa digunakan untuk membaca sebuah CD saja. Secara gari besar CD-ROM dibedakan menjadi 2 menurut tipenya yaitu : ATA/IDE  dan SCSI. Yang paling mendasari dari perbedaan tersebut adalah kecepatannya. Kalau ATA memiliki kecepatan 100-133Mbps sedangkan SCSI memiliki kecepatan kira-kira 150 Mbps. Untuk tipe SCSI biasanya ditemuka pada CR RW drive. Pada CD ROM terdapat tulisan 56X artinya kemampuan memberikan kecepatan transfer data sebesar 56 x150 Kbps. Tipe CD RW juga biasanya  dibedakan berdasarkan kemapuan membakar dan membaca. CD RW tipe 12x8x32 artinya memiliki kemampuan membakar pada CD R seccepat 12x, membakar pada CD RW secepat 8x, dan membaca CD R/CD RW/dengan kecepatan maksimal 32x.

40. hal yang membedakan antara CD-R dengan CD-RW


jika CD-R tidak bisa dihapus ataupun ditambah, jika sudah diisi file. CD - R hanya sekali pakai.
CD - RW sebaliknya.
-CD-R adalah singkatan dari Compact Disk Recordable adalah piringan pipih berbahan polikarbonat berdiameter 120 mm.  Data yang ditulis didalamnya bersifat WORM ( Write Once Read Many ) yang menunjukan bahwa data hanya dapat diisi sekali dan selanjutnya hanya bisa dilihat / dibaca.
-CD-RW yang merupakan pengembangan dari CD-R, Perbedaanya adalah CD-RW ( Compact Disk Recordable and reWriteable ) berbahan CD-RW menggunakan logam perpaduan antara perak, indium, antimon, dan telurrium untuk lapisan perekaman dan dapat diubah datanya berkali - kali, dapat di tambah, diganti atau dihapus sewaktu - waktu.
