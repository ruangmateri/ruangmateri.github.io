1. QuizContoh soal Dan Jawaban Dengan Rumus Materi Yang Sudah Tertera Di Gambar​


Soal

Sebuah transformator memiliki 300 lilitan primer dan 30 lilitan sekunder. Diketahui tegangan pada lilitan primer adalah 220 volt .

a. Tentukan besar tegangan pada lilitan sekunder

b. Jika arus listrik yang mengalir pada lilitan primer sebesar 0,5 mA berapakah arus yang mengalir pada lilitan sekunder

Penyelesaian :

[tex]a. \: \frac{Np}{Ns} = \frac{Vp}{Vs} \\ \\ \frac{300}{30} = \frac{220}{Vs } \\ \\ Vs = \frac{220 \times 30}{300} \\ \\ = 22 \: volt[/tex]

============

[tex]b. \: \frac{Np}{Ns} = \frac{Is}{Ip} \\ \\ \frac{300}{30} = \frac{Is}{0,5 \: mA} \\ \\ Is = \frac{300 \times 0,5 \: mA}{30} \\ \\ = 5 \: mA[/tex]

[tex]\fcolorbox{navy}{skyblue} {\purple{\boxed{\rm{ mas \: ryan \: < 3 }}}}[/tex]

Pendahuluan

Transformator (trafo) adalah komponen listrik yang digunakan untuk menaikan dan menurunkan tegangan arus listrik.

Transformator ada 2 jenis :

1) Transformator Step - Up (menaikan tegangan)

Ciri-ciri

a. Jumlah lilitan sekunder lebih banyak dari lilitan primer (NS > NP).

b. Tegangan sekunder lebih besar dari tegangan primer (VS > VP).

c. Kuat arus sekunder lebih kecil dari kuat arus primer (IS < IP).

2) Transformator Step - Down (menurunkan tegangan)

Ciri-ciri

a. Jumlah lilitan sekunder lebih sedikit dari lilitan primer (NS < NP).

b. Tegangan sekunder lebih kecil dari tegangan primer (VS < VP).

c. Kuat arus sekunder lebih besar dari kuat arus primer (IS > IP).

Hubungan antara jumlah lilitan dengan tegangan listrik ialah sebanding artinya, jika jumlah lilitan banyak Maka tegangannya besar dan sebaliknya. Rumus :

NP/NS = VP/VS

Hubungan antara jumlah lilitan tegangan dengan kuat arus listrik adalah berbanding terbalik artinya, jika jumlah lilitannya besar maka kuat arusnya kecil.

Rumus :

NP/NS = IS/IP dan VP/VS = IS/IP

Keterangan :

NP = Jumlah lilitan primer

NS = Jumlah lilitan sekunder

VP = tegangan primer

VS = Tegangan sekunder

IP = Arus primer

IS = Arus sekunder

Pembahasan Contoh soal

1) Diketahui :

NP = 800

NS = 200

IS = 0.3 A

VS = 110 V

Ditanya :

a. VP = ?

b. IS = ?

Jawab :

a. NP/NS = VP/VS

800/200 = VP/110

4/1 = VP × 110

VP = 110 × 4

VP = 440 V ✔️

b. NP/NS = IS/IP

800/200 = IS/0.3

4/1 = IS/0.3

IS = 4 × 0.3

IS = 1.2 A ✔️

15. Soal sudah tertera di gambar.​


" Himpunan "

_____________

M = { x | x ≤ 9, x ∈ Bil. Ganjil }

N = { x | 2 ≤ x < 11, x ∈ Bil. Prima }

______

>>Penyelesaian :

M = { 1, 3, 5, 7, 9 }

N = { 2, 3, 5, 7, }

M ∩ N = { 3, 5, 7 }

_________

Himpunan bagian dari M ∩ N diketahui dengan 2 cara :

- Mendaftarkan anggotanya :

Himpunan M ∩ N yang memiliki :

• 0 anggota : { }

• 1 anggota : { 3 }, { 5 }, dan { 7 }

• 2 anggota : { 3, 5 }, { 3, 7 }, dan { 5, 7 }

• 3 anggota : { 3, 5, 7 }

Total Himpunan Bagian :

>>1 + 3 + 3 + 1

>> 8 himpunan bagian

[tex] \\ [/tex]

• Menggunakan rumus :

[tex] \boxed{ \sf {2}^{n} }[/tex]

Dengan n = banyaknya anggota pada himpunan

>>2³

>> 2 × 2 × 2

>> 8 himpunan bagian

Penjelasan dengan langkah-langkah:

M = {x|x ≤9, x ∈ bilangan ganjil}

M = {1, 3,5,7,9}

--

N = {x|2≤x<11, x ∈ bilangan prima}

N = {2, 3,5,7}

--

MnN = {3, 5,7}

--

banyak himpunan bagian dari MnN = 2³ = 8

Maka, banyak himpunan bagian dari MnN adalah 8

27. Tertera Di gambar soalnya ​


Jawab:
(a.) Daerah asal {x|x < 3, x>3, x∈R}
(b.) Daerah asal {x|x < 3, x>3, x∈R}
(c.) Daerah asal {x|x < 3, x>3, x∈R}
(d.) Daerah asal {x|x < 0, x>0, x∈R}

Penjelasan :
Diketahui fungsi berikut
f(x) = x + 2
g(x) = x/(x-3)  {x≠3}

Ditanya daerah asal
(a.) (f+g)(x) =
f(x) + g(x) = x + 2 + x/(x-3)
Perhatikan penyebutnya
(x-3), daerah asal (x) tidak
boleh 0, x-3 ≠ 0, x ≠ 3
Daerah asal {x|x < 3, x>3, x∈R}

(b.) (f-g)(x) =
f(x) - g(x) = x + 2 - x/(x-3)
Sama seperti (a.) penyebutnya
(x-3), daerah asal (x) tidak
boleh 0, x-3 ≠ 0, x ≠ 3
Daerah asal {x|x < 3, x>3, x∈R}

(c.) (f · g)(x) =
f(x) · g(x) = (x + 2) (x/(x-3))
Dikali pembilangnya tidak
mengubah penyebutnya.
Penyebutnya tetap (x-3),
daerah asal (x) tidak
boleh 0, x-3 ≠ 0, x ≠ 3
Daerah asal {x|x < 3, x>3, x∈R}

(d.) (f/g)(x) =
f(x) ÷ g(x) = (x + 2) ÷ (x/(x-3))
= (x + 2) · ((x-3)/x)
Disini penyebutnya x
maka x tidak boleh 0
Daerah asal {x|x < 0, x>0, x∈R}

31. Soal tertera di foto​


Car is the Expensive vehicle

Mobil adalah kendaraan yang mahal

Kenapa jawabannya Expensive karena tidak dibandingkan dengan hal lain

Pembahasan :

3 Degress of Comparison

Positive degree

positive degree digunakan untuk menunjukkan bahwa hal yang dibandingkan di level yang sama atau sebanding

Keyword :

As... As

BMW is As Expensive As Audi

Comparative Degree

comparative degree digunakan untuk membandingkan sesuatu memiliki sifat lebih dibandingkan hal lain

Keyword :

Perbandingan 2 orang / hal

Adjective - ER or More + Adjective / Than

John is Taller than Andreas

Superlative Degree

superlative degree yang digunakan untuk membandingkan tiga atau lebih orang benda maupun beberapa hal

Keyword :

Ditambah :

■ ST / EST

■ The most + adjective

I am the shortest girl among my friends
