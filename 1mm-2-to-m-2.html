10. Sebuah celah berjarak 1mm disinari cahaya monokromatik dengan gelombang 2x10^7 M. Garis gelap terang dapat diamati pada layar yang berjarak 1 M dari celah. Hitunglah celah antara gelap ke-2 dan terang ke-4!


JAWABAN :

 

Dari soal dapat diketahui bahwa:

- jarak elah layar --> l = 1 m

- lebar celah --> d = 1 mm = 10^-3 m

- panjang gelombang --> λ = 2. 10^-7 m

 

Karena yang ditanyakan jarak pita gelap kedua dengan terang keempat, maka perlu mencari nilai n baru mencari nilai p.

 

Mula – mula kita cari n untuk garis gelap kedua dengan garis terang keempat.

g2 – t2 – g3 – t3 – g4 – t4 (5 interval x 0,5 maka n = 2,5)

 

Untuk mencari nilai p, digunakan rumus berikut:

 

nλ = d.p/l

1.     [tex] 2. 10^{-7}= \frac{ 10^{-3} . p }{1} [/tex]

p = 2. 10^-4 m = 0,2 mm

 

Berdasarkan uraian di atas, didapat bahwa jarak terdekat antara garis gelap kedua dengan garis terang keempat 0,2 mm.

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:

nλ = d.p/l

 

atau

nλ = d sin θ

 

keterangan:

n = orde terang/ gelap

λ = panjang gelombang

d = jarak celah

p = jarak antar pola

l = jarak layar ke celah

θ = sudut deviasi

 

A.      Interferensi celah ganda/ Young

Pola terang: n = 1, 2, 3, 4, dst..

Pola gelap: n = ½, 3/2, 5/2, 7/2, dst.

 

B.       Difraksi celah tunggal

Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.

Pola gelap: n = 1, 2, 3, 4, dst..

 

C.       Difraksi kisi

Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N

N = 1/d

17. panjang kawat 2 m diameter 0 5 mm hambatan 6 ohm hambatan kawat sejenis 2m dan diameter 1mm​


Hambatan kawat tersebut adalah 3 x 10⁻⁶. Untuk menentukan hambatan suatu kawat, kita gunakan persamaan:

R = ρL/A

Penjelasan dengan langkah-langkahDiketahui: Panjang kawat 2m, diameter 0,5 mm, hambatan 6 ohm, hambatan kawat sejenis 2m, dan diameter 1mmDitanya: Tentukan berapa ohm hambatan kawat tersebut (hambatan kawat kedua)?Jawab:

Langkah 1

Persamaan untuk menentukan hambatan kawat adalah:

R = ρL/A

Dengan:

R = hambatan kawat (ohm)

ρ = hambatan jenis kawat

L = panjang kawat (m)

A = luas penampang kawat

Langkah 2

Pada soal tersebut, kawat pertama:

L₁ = 2 m

d₁ = 0,5 mm, maka r₁ = 0,25 mm

Maka, R₁ = ρL₁/A₁

6 ohm = ρ (2)/ πr²

        6 = 2ρ/ 3,14 x (0,25 x 10⁻³)²

        6 =  2ρ/ 0,785 x 10⁻⁶

      2ρ = 4,71 x 10⁻⁶

        ρ = 2,355 x 10⁻⁶

sehingga kita dapatkan hambatan jenis kawat pertama sebesar 2,355 x 10⁻⁶

Langkah 3

Kawat kedua:

L₂ = 2 m

d₂ = 1 mm, maka r₂ = 0,5 mm

R₂ = ρL₂/A₂

    = (2,355 x 10⁻⁶) (2)/ 3,14 x (0,5 x 10⁻³)²

    = 4,71/ 3,14 x (0,5 x 10⁻⁶)

    = 4,71 / 1,57 x 10⁻⁶

    = 3 x 10⁻⁶
