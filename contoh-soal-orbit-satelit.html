6. Dua buah satelit A dan B mengorbit sebuah planet yang sama dengan jari-jari orbitnya masing-masing berurutan R dan 2R. Jika kecepatan orbit satelit A adalah v maka kecepatan orbit satelit B adalah


Jika kecepatan orbit satelit A adalah v maka kecepatan orbit satelit B adalah ¹/₂√2 v

[tex]\texttt{ }[/tex][tex]\texttt{ }[/tex]

PEMBAHASAN

Mari kita ingat kembali rumus Gaya Gravitasi sebagai berikut ini:

[tex]\texttt{ }[/tex][tex]\boxed{F = G\ \frac{m_1 m_2}{R^2}}[/tex]

dimana:

F = Gaya Gravitasi ( N )

G = Konstanta Gravitasi ( 6,67 x 10⁻¹¹ Nm² / kg² )

m = massa benda ( kg )

r = jarak antar benda ( m )

Mari kita sekarang mencoba menyelesaikan soalnya.

[tex]\texttt{ }[/tex][tex]\texttt{ }[/tex]

Diketahui:

Jari - jari orbit satelit A = R₁ = R

Jari - jari orbit satelit B = R₂ = 2R

Kecepatan satelit A = v₁ = v

Ditanyakan:

Kecepatan satelit B = v₂ = ?

Penyelesaian:

Kita akan menggunakan Hukum Gravitasi dan Gaya Sentripetal sebagai berikut ini:

[tex]\texttt{ }[/tex][tex]F = m \frac{v^2}{R}[/tex]

[tex]\texttt{ }[/tex][tex]G\ \frac{m M}{R^2} = m \frac{v^2}{R}[/tex]

[tex]\texttt{ }[/tex][tex]G\ \frac{M}{R} = v^2[/tex]

[tex]\texttt{ }[/tex][tex]\boxed{v = \sqrt{G\ \frac{M}{R}}}[/tex]

[tex]\texttt{ }[/tex][tex]\texttt{ }[/tex]

Selanjutnya dengan rumus diatas kita akan bandingkan kecepatan satelit A dan B sebagai berikut ini:

[tex]\texttt{ }[/tex][tex]v_1 : v_2 = \sqrt{G\ \frac{M}{R_1}} : \sqrt{G\ \frac{M}{R_2}}[/tex]

[tex]\texttt{ }[/tex][tex]v_1 : v_2 = \sqrt{\frac{1}{R_1}} : \sqrt{\frac{1}{R_2}}[/tex]

[tex]\texttt{ }[/tex][tex]v_1 : v_2 = \sqrt{R_2} : \sqrt{R_1}[/tex]

[tex]\texttt{ }[/tex][tex]v : v_2 = \sqrt{2R} : \sqrt{R}[/tex]

[tex]\texttt{ }[/tex][tex]v : v_2 = \sqrt{2}[/tex]

[tex]\texttt{ }[/tex][tex]v_2 = v \div \sqrt{2}[/tex]

[tex]\texttt{ }[/tex][tex]\boxed{v_2 = \frac{1}{2}\sqrt{2} v}[/tex]

[tex]\texttt{ }[/tex][tex]\texttt{ }[/tex]

12. Kelajuan satelit A mengorbit planet bernilai 3 kali kelajuan satelit B. Apabila jari-jari orbit satelit A sebesar R, jari-jari orbit satelit B adalah


Kelajuan satelit A mengorbit planet bernilai 3 kali kelajuan satelit B. Apabila jari-jari orbit satelit A sebesar R, jari-jari orbit satelit B adalah 9 R.

[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:7.\:HUKUM\:NEWTON\:(GAYA\:GRAVITASI)}}[/tex]

Gaya Gravitasi  merupakan gaya tarik menarik antara dua benda bermassa.

Persamaan Gaya Gravitasi adalah :

[tex]\boxed{F=G\frac{mM}{R^{2}}}[/tex]

Percepatan gravitasi merupakan percepatan suatu benda akibat ditarik benda bermassa lainnya.

Persamaan percepatan gravitasi adalah :

[tex]\boxed{g=G\frac{M}{R^{2}}}[/tex]

Periode orbit suatu satelit terhadap planet :

[tex]\boxed{T=2\pi \sqrt{\frac{r^{3} }{GM}}}[/tex]

Kecepatan Orbit satelit :

[tex]\boxed{v=\sqrt{\frac{GM}{r} }}[/tex]

Keterangan :

F : Gaya gravitasi (N)

g : Percepatan gravitasi benda (m/s²)

m : Massa benda (kg)

M : massa benda (kg)

R : Jari-jari benda (m)

r : Jari-jari orbit (m)

T : Periode orbit (s)

G : konstanta universal gravitasi (6,674 x 10⁻¹¹ Nm²/kg²)

v : kecepatan orbit

[tex]\boxed{Diketahui}[/tex]

Kecepatan orbit satelit A (vA) = 3 kali kecepatan orbit satelit B (vB)

Jari-jari orbit satelit A (rA) = R

[tex]\boxed{Ditanya}[/tex]

Jari-jari orbit satelit B

[tex]\boxed{Penyelesaian}[/tex]

Persamaan kecepatan orbit adalah :

v=\sqrt{\frac{GM}{r} }

v² ≈ 1/r

sehingga:

[tex]\frac{v_{A}^{2}}{v_{B}^{2}}=\frac{r_{B} }{r_{A} }[/tex]

[tex]r_{B}=\frac{v_{A}^{2}}{v_{B}^{2}} \:r_{A}[/tex]

dimana vA = 3 VB, maka :

[tex]r_{B}=\frac{(3v_{B})^{2}}{v_{B}^{2}} \:R[/tex]

[tex]r_{B}=\frac{9v_{B}^{2}}{v_{B}^{2}} \:R[/tex]

[tex]r_{B}=9 \:R[/tex]

Jadi jari-jari orbit satelit B adalah 9 R.

[tex]\boxed{Kesimpulan}[/tex]

Jari-jari orbit satelit B adalah 9 R.

14. 12. Dua buah satelit A dan B mengorbit sebuah planet yang sama, dengan jari jari orbit masing masing berurutan R dan 2R. Bila kecepatan orbit satelit A adalah v, maka kecepatan orbit satelit B adalah ... A. V​


Kecepatan orbit satelit dalam orbit melingkar terkait dengan jarak dan periode orbitnya. Dalam hal ini, karena satelit A dan B mengorbit planet yang sama, kita dapat menggunakan hukum Kepler untuk menghubungkan kecepatan orbit mereka.

Hukum Kepler menyatakan bahwa kuadrat periode orbit (T) suatu objek berbanding lurus dengan kuasa tiga dari jarak orbit (r) objek tersebut terhadap pusat yang diorbitinya.

Dalam hal ini, satelit A memiliki jari-jari orbit R, sehingga kecepatan orbitnya (v) dapat dinyatakan sebagai:

v = 2πR/T

Satelit B memiliki jari-jari orbit 2R. Kita ingin mencari kecepatan orbitnya (v'). Dalam hal ini, periode orbit satelit B (T') berhubungan dengan jarak orbitnya:

T' = k(2R)^3

Karena jarak orbit satelit B dua kali lebih besar dari satelit A (2R = 2 * R), maka kuadrat periode orbitnya (T') akan berhubungan dengan kuasa tiga jarak orbit satelit A (R) sebagai berikut:

T' = k(2R)^3 = k(2^3 * R^3) = 8kR^3

Dengan demikian, kecepatan orbit satelit B (v') dapat dinyatakan sebagai:

v' = 2π(2R)/T' = 2π(2R)/(8kR^3) = π/(4kR^2)

Jadi, kecepatan orbit satelit B adalah π/(4kR^2).

17. Dua satelit P dan Q mengorbit planet yang sama. Jari-jari orbit satelit P dan Q adalah sebesar 2R dan 3R. Jika kecepatan orbit satelit P adalah v, maka kecepatan orbit satelit Q adalah​


Kecepatan orbit satelit Q : (1/3√6) v

Penjelasan:

Dasar Teori

Satelit yang mengorbit sebuah planet memenuhi persamaan gerak melingkar beraturan (anggap kecepatan orbit satelit konstan):

Σ F = Fs

Gaya yg bekerja pada benda adalah gaya gravitasi planet dan satelit yang dinyatakan

Fg = G ms Mp/r^2

Fs adalah gaya sentripetal yg dialami satelit, dinyatakan

Fs = ms Vs^2/r

sehingga :

G ms Mp/r^2 = ms Vs^2/r

Kecepatan orbit satelit:

19. Dua buah satelit A dan B mengorbit sebuah planet yang sama, dengan jari jari orbit masing masing berurutan R dan 2R. Bila kecepatan orbit satelit A adalah v, maka kecepatan orbit satelit B adalah ... A. V​


Jawaban: Jadi, kecepatan orbit satelit B (v_B) adalah 4 kali kecepatan orbit satelit A (v). Dengan kata lain, v_B = 4v.

Penjelasan: Kecepatan orbit satelit dapat dihitung menggunakan hukum kekekalan momentum sudut. Dalam hal ini, kita dapat menggunakan hukum Kepler tentang periode orbit.

Hukum Kepler menyatakan bahwa kuadrat periode orbit satelit (T) berbanding terbalik dengan kuasa tiga dari jarak orbit (R):

T² ∝ R³

Karena satelit B memiliki jarak orbit dua kali lipat dari satelit A (2R), kita dapat menulis perbandingan periode orbitnya sebagai:

T_B² / T_A² = (2R)³ / R³

T_B² / T_A² = 8

Dalam hal ini, karena satelit A memiliki kecepatan orbit v, periode orbit satelit A dapat dituliskan sebagai:

T_A = 2πR / v

Menggabungkan persamaan di atas, kita dapat menulis:

T_B² / (2πR / v)² = 8

T_B² = 8 * (2πR / v)²

T_B = 2 * (2πR / v)

T_B = 4πR / v

Jadi, kecepatan orbit satelit B (v_B) adalah 4 kali kecepatan orbit satelit A (v). Dengan kata lain, v_B = 4v.

31. Dua satelit x dan y mengorbit sebuah planet dengan perbandingan jari jari orbit 16 : 81 apabila kelajuan orbit satelit x sebesar v,kelajuan orbit satelit y sebesar


Diketahui : 
[tex] r_{x} : r_{y}=16:81[/tex]
[tex] v_{x}=v[/tex]

Ditanya :
[tex] v_{y}=...?[/tex]

Penyelesaian :
Kelajuan orbit satelit dirumuskan sebagai
[tex]v= \sqrt{ \frac{GM}{R} } [/tex]

untuk memperoleh berapa besar [tex] v_{y}[/tex], kita perlu melakukan perbandingan [tex] v_{x}[/tex] dan [tex] v_{y}=...?[/tex]

Perhatikan uraian berikut.
[tex] \frac{ v_{x} }{ v_{y} } = \frac{ \sqrt{ \frac{GM}{ r_{x} } } }{ \sqrt{ \frac{GM}{ r_{y} } } } [/tex]
[tex] \frac{v}{ v_{y} } = \frac{ \sqrt{ \frac{1}{ r_{x} } } }{ \sqrt{ \frac{1}{ r_{y} } } } [/tex]
[tex] \frac{v}{ v_{y} } = \sqrt{ \frac{ r_{y} }{ r_{x} } } [/tex]
[tex] \frac{v}{ v_{y} } = \sqrt{ \frac{81}{16} } [/tex]
[tex] \frac{v}{ v_{y} } = \frac{9}{4} [/tex]
[tex]9 v_{y} =4v[/tex]
[tex] v_{y} = \frac{4}{9}v [/tex] m/s
