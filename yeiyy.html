› Permutasi

Permutasi adalah penyusunan kembali suatu kumpulan objek dalam urutan yang berbeda dari urutan yang semula. Rumus permutasi yaitu;

Jika terdapat unsur/huruf ganda, maka rumusnya:[tex]\tt \large{P = \frac{n!}{k!}}[/tex]

Dan jika tidak terdapat unsur/huruf ganda, maka rumusnya:[tex]\tt \large{P = n!}[/tex]

Dengan:

P = Permutasi/banyak susunan

n = jumlah unsur/huruf

k = unsur/huruf ganda

! = faktorial

Tabung merupakan bangun sisi lengkung yang memiliki bidang alas dan bidang atas berbentuk lingkaran yang sejajar dan kongruen.

Ciri-ciri tabung adalah :

Mempunyai 3 sisi yaitu, alas, selimut,dan tutupMempunyai 2 rusukTidak mempunyai titik sudutBidang alas dan tutup berbentuk lingkaran yang sama besar

[tex]\boxed{\tt{\blue{= 2πr × t}}}[/tex]

↬ Rumus mencari luas permukaan tabung adalah :

[tex]\boxed{\sf{\green{= 2πr (t + r)}}}[/tex]

PENDAHULUAN :

Tabung adalah bangun ruang yang dibatasi oleh sebuah bidang lengkung ( selimut tabung ) dan dua bidang datar ( sisi alas dan Sisi atas ) sisi alas dan Sisi atas berbentuk lingkaran yang letaknya sejajar.

Ciri - Ciri Tabung

( A ). mempunyai sisi atas ( tutup ) dan Sisi bawah ( alas ) yang berbentuk lingkaran.

( B ). mempunyai sisi samping ( sisi lengkung ) disepanjang tingginya.

A. Luas Selimut Tabung

Selimut tabung sisi lengkung keliling lingkaran bawah alas sama dengan keliling lingkaran atas tutup :

L.s.t = panjang selimut tabung × lebar selimut tabung

L.s.t = keliling lingkaran alas kali tinggi tabung L.s.t = 2πr ( t + r )

Luas Selimut Tabung Dirumuskan Sebagai Berikut :

Luas Selimut Tabung : 2πr × t

dengan r menyatakan jari-jari tabung, t menyatakan tinggi tabung dan nilai π = 3,14 atau π = 22/7

B. Luas Permukaan Tabung

Permukaan tabung terdiri atas selimut tabung sisi atas ( tutup ) dan Sisi bawah alas Maka luas permukaan tabung dirumuskan sebagai berikut karena :

L.p.t = luas selimut tabung + luas sisi atas

L.p.t = ( tutup ) + luas sisi bawah ( alas )

L.p.t = ( 2πr × t ) + πr² + πr²

L.p.t = ( 2πr × t ) + 2πr²

L.p.t = 2πr ( t + r )

Luas seluruh permukaan tabung dirumuskan sebagai berikut.

Luas Permukaan tabung : 2πr × ( t + r )

dengan r menyatakan jari-jari tabung, t menyatakan tinggi tabung dan nilai π = 3,14 atau π = 22/7

C. Volume Tabung

Volume tabung terdiri atas alas prisma tutup Prisma akan berbentuk prisma yang mempunyai bentuk alas tutup berupa lingkaran disebut tabung. Oleh karena itu diperoleh volume tabung sebagai berikut :

Vt = Luas alas tabung × tinggi tabung

Vt = Luas lingkaran × tinggi tabung

Vt = ( πr² ) × t

Vt = πr² t

Volume tabung dapat di rumuskan sebagai berikut :

Volume Tabung : πr²t

dengan r menyatakan jari-jari tabung, t menyatakan tinggi tabung dan nilai π = 3,14 atau π = 22/7
