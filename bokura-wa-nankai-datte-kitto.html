4. arti daijobou desu datte kimi yoaimo​


'Daijoubu desho, datte kimi yowai mon' artinya adalah "Gapapa kan, soalnya kamu lemah sih".

-

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN

❌ Daijobou desu datte kimi yoaimo

⭕ Daijoubu desho datte kimi yowai mon

Kalimat di atas merupakan kalimat yang diucapkan oleh salah satu karakter dari anime Jujutsu Kaisen, bernama Satoru Gojo. Kalimat ini belakangan sangat terkenal, terutama di kalangan wibu... eh, anime lovers (ehe :3). Penjelasan tiap kata dari kalimat tersebut adalah sebagai berikut.

---

大丈夫 (Daijoubu)

Artinya adalah "Tidak apa-apa" atau "Gapapa". Kata ini termasuk ke dalam kelompok kata sifat-na, yaitu kumpulan kata sifat yang akhiran huruf dari kata nya beragam.

-

でしょ (Desho)

Artinya seperti akhiran "-kan?". Tujuannya seperti untuk meminta persetujuan dari lawan bicara tentang hal yang sedang dibahas.

-

だって (Datte)

Artinya yaitu "Karena", "Soalnya", "Abisnya", dsb. Ini merupakan partikel, yang digunakan sebagai awalan kalimat untuk menyatakan alasan.

-

君 (Kimi)

Artinya adalah "Kamu". Kata ini merupakan kata ganti orang kedua, yang biasa digunakan ke orang yang setara atau lebih muda. Kata ini tidak digunakan ke orang yang lebih tinggi atau lebih tua dari kita.

-

弱い (Yowai)

Artinya adalah "Lemah". Kata ini termasuk ke dalam kelompok kata sifat-i, yaitu kumpulan kata sifat yang akhiran huruf dari kata nya adalah huruf 'i'.

-

もん (Mon)

Tidak memiliki arti khusus. Kata ini merupakan salah satu tata bahasa Jepang, yang gunanya untuk menunjukkan suatu alasan. Diletakkan setelah predikat, yang merupakan alasan dari hal yang dibahasnya. Biasa jika menggunakan kata ini, di awal kalimatnya terdapat kata 'datte'.

7. Daijoubu desu datte kimi yowai mo artinya ​


Kalimat 'daijoubu deshou, datte kimi yowai mon' berarti gapapa kan, karena kamu lemah sih.

Pembahasan

Salah satu kalimat Gojo Satoru yang merupakan salah satu karakter protagonis dalam Jujutsu Kaisen ketika pada saat melawan musuhnya dan meremehkan musuhnya tersebut.

-------

Daijoubu「大丈夫・だいじょうぶ」

Untuk arti ialah tidak/gak apa-apa, baik-baik saja.

Deshou「でしょう」

Kata ini pengucapannya termasuk bentuk kata dan pengucapan secara sopan, dan jika dalam bahasa Indonesia memiliki arti bukan?, kan?, mungkin (arti tergantung kalimatnya).

Datte「だって」

Untuk artinya dapat berarti karena, habisnya. Dapat digunakan untuk ungkapan yang ingin menyampaikan suatu alasan.

Kimi「君・きみ」

Termasuk golongan kata ganti yang informal, untuk arti diartikan sebagai kamu/kau. Pemakaian kata kimi ini hampir diucapkan bagi laki-laki, orang-orang yang cocok dipanggil dengan kata ganti kimi adalah untuk orang yang seumuran, yang masih muda (anak atau remaja), bawahan/anak buah, dan terhadap orang yang akrab.

Yowai「弱い・よわい」

Artinya adalah lemah. Kata dengan kelas kata sifat i-keiyoushi (akhiran -i).

Mon「もん」

Umum diucapkan oleh pria, dan untuk fungsinya sendiri yaitu untuk menyampaikan atau menyatakan hal alasan mengenai alasan yang disampaikan pada kalimat.

20. apa arti dari "datte" ?​


Pembahasan

Kata datte ini memiliki beberapa arti seperti :

Karena.Bahkan.Pun.Habisnya (...).

Kata datte ditulis dalam huruf Jepangnya adalah だって yang ditulis dalam huruf hiragana.

__________________________

Jika kata だって (datte) diartikan sebagai bahkan atau pun maka umumnya kata ini menjadi suatu partikel (adverbia). Umumnya jika bertujuan sebagai partikel adverbia, maka tentunya ada pola sebagai berikut :

bentuk kt. benda/kt. tanya + だって (datte) + predikat.

_____

Jika menyatakan arti karena, habisnya (...) umumnya merupakan suatu partikel datte yang digunakan untuk menyatakan suatu alasan atau sebab yang terjadi. Jika untuk menyatakan suatu sebab, umumnya menggunakan pola sebagai berikut :

だって (datte) + jenis alasan/sebab.

Tentunya ketika menyatakan suatu alasan, harus menggunakan awalan kata yaitu だって (datte).

22. Apa arti dari "Daijoubu desu Datte kimi Yowai mo?"​


大丈夫です。だって君弱いも (daijoubu desu, datte kimi yowai mo) artinya adalah seharusnya tidak apa-apa, karena kamu itu lemah.

Pembahasan

Tentunya, kalian mendengar kalimat ini dari anime atau manga ya kan? Salah satu anime Jujutsu Kaisen, dengan seorang tokoh bernama Gojo Satoru yang salah satunya pernah mengucapkan kalimat ini ke musuh/lawannya.

-> 大丈夫 (daijoubu) artinya tidak apa-apa, ini merupakan jenis bentuk kata sifat golongan II. Kata ini bertujuan untuk mengungkapkan ketidak-raguan atau merasa aman dari situasi tertentu. Misalnya seperti Gojo Satoru yang menyebutkan kata ini dikarenakan dia sendiri tidak merasa ragu terhadap dirinya sendiri dan sekaligus meremehkan musuh yang dilawannya.

-> です (desu) merupakan jenis kata penegasan atau kata yang terkadang diletakkan setelah kata sifat atau benda. Walau kata ini merupakan kata penegasan namun ini tetap sopan diucapkan. Jika kita ingin mengucapkan kata "aku tidak apa-apa" maka kita menambahkan daijoubu + desu sehingga

menjadi 大丈夫です (daijoubu desu).

-> だって (datte) artinya ke dalam bahasa Indonesia adalah karena.

-> 君 (kimi) artinya ke dalam bahasa Indonesia adalah kamu. Kata ini merupakan bentuk kata yang informal (bukan kata resmi/formal).

-> 弱い (yowai) artinya adalah lemah, selain lemah ada juga arti lainnya seperti tidak mahir atau payah.

-> も (mo). Ini merupakan partikel, yang bisa diartikan sebagai "pun, juga, bahkan, dan." Polanya bisa diletakkan sebelum subjek = も + subjek, atau diletakkan di akhir objek = objek + も .

25. arti "daijoubu desu datte kimi yowaimo"​


Arti kalimat 'daijoubu desu, datte kimi yowai mon' yaitu 'gapapa, habisnya kamu lemah sih.'

Pembahasan

Daijoubu desu, datte kimi yowai mon

「大丈夫です。だって君弱いもん」

Untuk terjemahan dan penjelasan mengenai setiap kata-perkatanya ada dibawah sebagai berikut :

Daijoubu「大丈夫」: tidak apa-apa, gak apa-apa, baik-baik saja. Ungkapan menyatakan suatu ketidak khawatiran.

Desu「です」 : hanya untuk fungsi penegas dalam kalimatnya diatas, dan tidak mempunyai arti.

Datte「だつて」: yang berarti karena, soalnya. Jenis partikel, yang dimana dalam kalimat ini berfungsi untuk menyatakan suatu alasan.

Kimi「君 • きみ」: kamu, penggunaan kata ini hanya ketika memanggil orang yang rendah, dibawah, masih muda.

Yowai「弱い」: berarti lemah, gak mampu, gak kuat. Menyatakan hal yang tidak dapat dilakukan/tidak mampu.

Mon「もん」: sejenis golongan predikat, penggunaannya termasuk kasual. Predikat ini diletakkan dalam kalimat yang berisikan mengenai menyatakan suatu alasan.
