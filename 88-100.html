32. Suku barisan ke-9 dari 100, 95, 88, 79, ... adalah ... .​


Jawaban:

100, 92, 85, 79, 74, 70

Untuk mengetahui cara mendapatkan hasil dapat disimak dalam pembahasan berikut.

Pembahasan

Soal ini berkaitan dengan barisan aritmetika bertingkat. Barisan aritmetika adalah barisan yang memiliki selisih yang selalu tetap/sama. Rumus suku ke-n barisan aritmetika bertingkat adalah

Un = a + (n-1) b + (n-2)(n-1) c/2 + ...

Oke, langsung saja kita membahas persoalan.

Penyelesaian

100      92         ...         79    ....      70 ⇒ a = 100

       -8       ...          ...        ...       ...      ⇒ b = -8

            ---        ...        ...         ...           ⇒ c?

Sebelum mencari suku pada barisan yang hilang kita perlu menentukan rumus suku ke-n nya dengan langkah berikut:

Langkah 1: Mencari nilai c

Un = a + (n - 1) b + (n - 2)(n - 1) c/2

U₄ = 100 + (4 - 1) (-8) + (4 - 2)(4 - 1) c/2

79 = 100 + 3 (-8) + 2 . 3 . c / 2

79 = 100 - 24 + 3c

79 = 76 + 3c

3c = 79 - 76

3c = 3

c = 1

Langkah 2: Menentukan rumus suku ke-n

Un = a + (n - 1) b + (n - 2)(n - 1) c/2

Un = 100 + (n - 1)(-8) + (n - 2)(n - 1) 1/2

Un = 100 - 8n + 8 + 1/2 (n² - 3n + 2)

Un = -8n + 108 + 1/2 n² - 3/2 n + 1

Un = 1/2 n² - 19/2 n + 109

Sekarang, kita tinggal mencari suku ke-3 dan suku ke-5 dengan menggunakan rumus Un

Un = 1/2 n² - 19/2 n + 109

U₃ = 1/2 (3)² - 19/2 (3) + 109

U₃ = 9/2 - 57/2 + 109

U₃ = -48/2 + 109

U₃ = -24 + 109 = 85

U₅ = 1/2 (5)² - 19/2 (5) + 109

U₅ = 1/2 (25) - 95/2 + 109

U₅ = 25/2 - 95/2 + 109

U₅ = -70/2 + 109

U₅ = -35 + 109 = 74

Maka, angka/suku yang hilang adalah 85 dan 7
