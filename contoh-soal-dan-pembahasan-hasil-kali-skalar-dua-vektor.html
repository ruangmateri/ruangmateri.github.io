34. Buatlah soal dan jawaban tentang perkalian skalar dua vektor


Hitung terlebih dahulu QR, PR dan PQ

QR = R - Q = (1, 2, -2) - (-1, 2, 0) = (2, 0, -2)

PR = R - P = (1, 2, -2) - (-3, -1, -5) = (4, 3, 3)

PQ = Q - P = (-1, 2, 0) - (-3, -1, -5) = (2, 3, 5)

Hitung (QR + PR) = b

QR + PR = (2, 0, -2) + (4, 3, 3) = (6, 3, 1)

Jadi

a . b = PQ . (QR + PR) = (2, 3, 5) . (6, 3, 1) = (2 . 6) + (3 . 3) + (5 . 1) = 12 + 9 + 5

a . b = 26

Diketahui p = (-2, 0, -3) dan q = (5, -3, -3). Hasil kali skalar p dan q dapat dihitung dengan rumus berikut:

p • q = x1 • x2 + y1 • y2 + z1 • z2

Substitusi (x1, y1, z1) = (-2, 0, -3) dan (x2, y2, z2) = (5, -3, -3), diperoleh:

p • q = -2(5) + 0(-3) + (-3)(-3)

= -10 + 0 + 9

= -1

Hasil kali skalar p dan q adalah -1.
