34. Booting? Warm Boot? Cold Boot? Prosedur warm boot dan cold boot?


Booting adalah proses membaca seluruh hardware sampai dengan kondisi komputer siap untuk digunakan.
Warm Booting (Booting Panas) adalah Proses pengaktifan kembali computer dimana kondisi komputer saat itu menyala.
Prosedur Warm Booting sebagai berikut :
- Klik tombol Start
- Klik Restart pada menu pilihan Shutdown

Cold Boot (Booting dingin) adalah Proses pengaktifan / menghidupkan komputer dimana kondisi awal komputer dalam keadaan mati.
Prosedur cold booting sebagai berikut:
-  Pastikan bahwa kabel pada power supply/ listrik sudah terpasang dengan   baik.
-  Tekan tombol power pada di CPU

# semoga membantu
booting = tampilan ketika komputer dinyalakan
warm boot = merebot komputer melalu tombol dipencet langsung
cold boot = merebot komputer melalui prosedur windows

warm boot = tekan tombol restart di cpu
cold boot = klik start, klik panah dekat shutdown, klik restart
