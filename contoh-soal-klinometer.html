1. Apa itu klinometer ?


Klinometer adalah alat sederhana untuk mengukur sudut elevasi antara garis datar dan sebuah garis yang menghubungkan sebuah titik pada garis datar tersebut dengan titik puncak (ujung) sebuah objek. Aplikasinya digunakan untuk mengukur tinggi (panjang) suatu objek dengan memanfaatkan sudut elevasi.

2. fungsi dan kegunaan klinometer ?


untukmengukur suatu kecepatan


3. bagaimana sejarah penemuan klinometer?


Klinometer sudah ada sejak awal abad 20. Versi awal sangat bergantung pada bobot sebagai sarana untuk menentukan kemiringan dan jaraknya. Kemudian inkarnasi dari klinometer membuat penggunaan tabung kaca melengkung diisi dengan beberapa jenis cairan redaman dan bola baja untuk memetakan sudut dan lereng. Saat ini, penggunaan sensor elektronik merupakan komponen penting dalam desain dan fungsi dari klinometer modern.

4. 4. Pandu ingin mengetahui tinggi pohon pinus menggunakan klinometer. Pada pengamatan pertama, ujung pohon pinus terlihat pada klinometer menunjuk pada sudut 60°. Pada pengamatan kedua, ia bergerak menjauhi pohon pinus sejauh 24 meter dan terlihat pada klinometer sudut 30°. Tinggi pohon pinus tersebut adalah.... A. 30√3 m B. 24√3 m C. 20√3 m D. 20 m E. 12 m​


Tinggi pohon pinus tersebut adalah 12√3 m.

(Jawaban: Tidak ada di pilihan)

Nilai Trigonometri di Sudut Istimewa:

tan 30° = ¹/₃ √3

tan 45° = 1

tan 60° = √3

Penjelasan dengan langkah-langkah:

Diketahui:

Pada pengamatan pertama, ujung pohon pinus terlihat pada klinometer menunjuk pada sudut 60°.

Pada pengamatan kedua, ia bergerak menjauhi pohon pinus sejauh 24 meter dan terlihat pada klinometer sudut 30°.

Ditanya:

Tinggi pohon pinus tersebut adalah

Jawab:

Misalkan:

Jarak pertama Pandu ke pohon pinus = x

Jarak kedua Pandu ke pohon pinus = x + 24

Tinggi pohon pinus = h

tan 60° = [tex]\frac{h}{x}[/tex]

⇔ h = x × tan 60°

tan 30° = [tex]\frac{h}{x+24}[/tex]

⇒ h = (x + 24) × tan 30°

Karena tinggi pohon sama maka:

h = h

x tan 60° = (x + 24) × tan 30°

x × √3 = (x + 24) ×  ¹/₃√3

x = (x + 24) × ¹/₃

3x = x + 24

3x - x = 24

2x = 24

x = 12

Tinggi pohon:

h = x × tan 60°

h = 12 × √3

h = 12√3

Jadi, tinggi pohon pinus tersebut adalah 12√3 m.

9. seorang siswa diberikan tugas untuk mengukur tinggi sebuah gedung dengan menggunakan klinometer pada awal berdiri melihat ujung atas gedung terlihat jarum pada klinometer 45 derajat kemudian mendekati gedung sejauh 20 meter dan terlihat pada KM dengan sudut 60° tinggi gedung tersebut adalah


Seorang siswa diberikan tugas untuk mengukur tinggi sebuah gedung dengan menggunakan klinometer pada awal berdiri melihat ujung atas gedung terlihat jarum pada klinometer 45° kemudian mendekati gedung sejauh 20 meter dan terlihat pada KM dengan sudut 60°, tinggi gedung tersebut adalah (30 + 10√3) meter. Hasil tersebut diperoleh dengan menggunakan perbandingan trigonometri. Berikut rumus perbandingan pada trigonometri:

sin A = [tex]\frac{sisi \: depan}{sisi \: miring} = \frac{de}{mi}[/tex] cos A = [tex]\frac{sisi \: samping}{sisi \: miring} = \frac{sa}{mi}[/tex] tan A = [tex]\frac{sisi \: depan}{sisi \: samping} = \frac{de}{sa}[/tex]

Rumus Pythagoras:  

mi² = de² + sa²


Pembahasan

Misal  

x = jarak anak ke gedung mula-mula

maka

tan 45° = [tex]\frac{tinggi \: gedung}{jarak \: anak \: ke \: gedung}[/tex]

1 = [tex]\frac{tinggi \: gedung}{x}[/tex]

x = tinggi gedung


Seorang anak mendekati gedung sejauh 20 meter, sehingga jarak anak sekarang ke gedung adalah (x – 20) m

tan 60° = [tex]\frac{tinggi \: gedung}{jarak \: anak \: ke \: gedung}[/tex]

√3 = [tex]\frac{x}{x - 20}[/tex]

√3 (x – 20) = x

x√3 – 20√3 = x

x√3 – x = 20√3

x(√3 – 1) = 20√3

x = [tex]\frac{20 \sqrt{3}}{\sqrt{3} - 1}[/tex]

x = [tex]\frac{20 \sqrt{3}}{\sqrt{3} - 1} \times \frac{\sqrt{3} + 1}{\sqrt{3} + 1} [/tex]

x = [tex]\frac{20 \sqrt{3} (\sqrt{3} + 1)}{3 - 1}[/tex]

x = [tex]\frac{20 \sqrt{3} (\sqrt{3} + 1)}{2}[/tex]

x = 10√3 (√3 + 1)

x = (30 + 10√3)


Jadi tinggi gedung tersebut adalah (30 + 10√3) meter

Sudut z = 28°

Amir mengukur tinggi rumah dengan menggunakan klinometer. Pada tempat pertama Amir berdiri,ujung atas rumah terlihat pada klinometer dengan sudut 60°. Kemudian, Amir bergerak menjauhi rumah sejauh 4 meter dan ujung atas rumah terlihat pada klinometer dengan sudut 45°. Tinggi rumah tersebut adalah (6 + 2√3) m

Untuk mencari tinggi rumah maka kita menggunakan perbandingan Trigonometri segitiga siku-siku

Rumus Trigonometri :

Sin A = [tex]\frac{depan}{miring} [/tex]

Cos A = [tex]\frac{samping}{miring} [/tex]

Tan A = [tex]\frac{depan}{samping}[/tex]

Pembahasan :

Diketahui :

Pengukuran pertama :

sudut yang dibentuk 60°

Jarak Amir ke rumah x m

Pengukuran kedua :

Sudut yang dibentuk 45°

Jarak Amir ke rumah (x+4) m

Ditanya :

Tinggi rumah tersebut ?

Dijawab :

Untuk mencari tinggi rumah kita gunakan perbandingan Tangen

Pengukuran pertama :

PR = Tinggi rumah

RA = Jarak Amir dan rumah = x m

Sudut yang dibentuk 60°

Tan 60° = √3

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 60° = [tex]\frac{PR}{RA}[/tex]

√3 = [tex]\frac{PR}{x}[/tex]

PR = √3 x

Pengukuran kedua :

PR = Tinggi rumah

RA = Jarak Amir dan rumah = (x + 4) m

Sudut yang dibentuk 45°

Tan 45° = 1

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 60° = [tex]\frac{PR}{RA}[/tex]

1 = [tex]\frac{PR}{x + 4}[/tex]

PR = x + 4

Selanjutnya kita cari nilai x

PR = PR

√3 x = x + 4

x - √3 x = -4

x(1 - √3) = -4

x = [tex]\frac{-4}{1 - \sqrt{3}} \times \frac{1 + \sqrt{3}}{1 + \sqrt{3}}[/tex]

x = [tex]\frac{-4(1 + \sqrt{3})}{1 - 3}[/tex]

x = [tex]\frac{-4 - 4\sqrt{3}}{-2}[/tex]

x = 2 + 2√3

Selanjutnya kita Cari tinggi rumah

PR = x + 4

PR = 2 + 2√3 + 4

PR = (6 + 2√3) m

∴ Jadi tinggi rumah tersebut adalah (6 + 2√3) m

Cos A = [tex]\frac{samping}{miring}[/tex]

Tan A = [tex]\frac{depan}{samping}[/tex]

Pembahasan :

Diketahui :

Pengukuran pertama sudut = 30°

Jarak = x m

Pengukuran kedua sudut = 45°

Jarak = (x - 18) m

Ditanya :

Tinggi tiang listrik ?

Dijawab :

Untuk mencari tinggi tiang listrik kita gunakan perbandingan Tangen

PT = tinggi tiang listrik

TA = jarak anak ke tiang listrik= x m

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 30° = [tex]\frac{PT}{TA}[/tex]

Tan 30° = [tex]\frac{1}{3} \sqrt{3}[/tex]

[tex]\frac{1}{3} \sqrt{3} = \frac{PT}{x}[/tex]

Kita gunakan kali silang

√3 x = 3 PT

PT = [tex]\frac{1}{3} \sqrt{3}[/tex]x

Selanjutnya kita cari perbandingan yang kedua

PT = tinggi tiang listrik

TA = jarak anak ke tiang listrik = (x - 18) m

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 45° = [tex]\frac{PT}{TA}[/tex]

Tan 45° = 1

1 = [tex]\frac{PT}{x - 18}[/tex]

PT = x - 18

Selanjutnya kita cari nilai x

PT = PT

[tex]\frac{1}{3} \sqrt{3}[/tex] x = x - 18

x - [tex]\frac{1}{3} \sqrt{3}[/tex] x = 18

[tex]x(1 - \frac{1}{3} \sqrt{3})[/tex] = 18

[tex]x = \frac{18}{1 - \frac{1}{3} \sqrt{3}} \times \frac{1 +\frac{1}{3}\sqrt{3}}{1 + \frac{1}{3}\sqrt{3}}[/tex]

[tex]x = \frac{18(1+\frac{1}{3} \sqrt{3})}{1 - \frac{1}{3}}[/tex]

[tex]x = \frac{18 + 6\sqrt{3}}{\frac{2}{3}}[/tex]

x = (27 + 9√3) cm

Selanjutnya kita cari tinggi tiang listrik

PT = x - 18

PT = (27 + 9√3) - 18

PT = (9√3 + 9) m

∴ Jadi tinggi tiang listrik adalah (9√3 + 9) m

39. Tinggi menara 85m

Faktor skala 1 : 700

Penjelasan dengan langkah-langkah:

Diketahui:

Hasil pengamatan :

tinggi obyek = 12 cm

Alasnya = 5 cm

Titik bidik 1 m dari tanah

Jarak bidik sebenarnya 35 m

Dicari:

Ukuran menara dan skala

Jawab :

1. jarak titik pengamat ke menara = 35 m = 3500 cm

jarak ini adalah jarak alas

2. dengan perbandingan sebangun ditemukan faktor skala:

Alas : 5 : 3500 = 1 : 700

tinggi sebenarnya dari titik pengamatan : 12/1 x 700 cm = 8400cm

atau 84 meter.

Tinggi seluruhnya dari atas tanah:

= 84 + tinggi titik pengamatan = 84+1 = 85 m
