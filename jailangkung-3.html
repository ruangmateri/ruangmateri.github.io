2. ringkasan film jailangkung 2017


Film horor Indonesia berjudul “Jailangkung” ini merupakan film bercerita mengenai 3 orang gadis yang penasaran dan ingin membongkar sebuah misteri yang terjadi pada Ayah mereka yang ditemukan koma dengan misterius. Bella ( Amanda Rawles ) bersama dengan sang kakak Angel ( Hannah Al Rasyid ) dan juga adiknya Tasya mencoba mencari tahu.

Dengan bantuan dari teman kampus Bella, Rama ( Jefri Nichol ) yang juga tertarik dengan dunia mistik. Serta juga dari seorang pilot bernama Kapten Wardana ( Augie Fantinus ) yang membantu mereka untuk menemukan sebuah tempat di mana ditemukannya Ferdi ( Lukman Sardi ). Di tempat tersebut mereka terkejut dengan kemegahan yang dinamakan sebagai ‘Jailangkung’. Di mana Jailangkung membuka sebuah rahasia gelap di masa lalu dan masa kini yang membahayakan hidup mereka.

PASSIVE VOICE  

Kalimat aktif adalah kalimat yang subjeknya melakukan aksi, sementara kalimat pasif adalah kalimat dimana subjeknya tidak melakukan aksi tetapi menerima aksi. Ciri khas dari kalimat passive adalah TOBE+V3(Past Participle). Syarat-syarat perubahan kalimat aktive ke kalimat pasif adalah:  

Subjek pada kalimat aktif berubah menjadi objek pada kalimat pasif. Sebaliknya, objek pada kalimat aktif berubah menjadi subjek pada kalimat pasif.  Selalu dibentuk dengan menggunakan pola TOBE+ VERB 3 (Past particile).  Tobe yang digunakan sesuai dengan pola tenses dari kalimat aktif. Terkadang menggunakan kata by jika yang melakukan aksi/perbuatan pada kalimat aktif perlu dan sangat penting untuk diketahui.  Passive Voice pada beberapa tense:  

1. Simple Present Tense  

Aktif : S + V1+O  

Pasif : S + to be (am, are, is) + V3 + by + O  

2. Present Continuous Tense  

Aktif : S + to be (am, are, is) + V1-ing + O  

Pasif : S + to be (am, are, is) + being + V3 + by + O  

3. Present Perfect Tense  

Aktif : S + have/has + V3 + O  

Pasif : S + have/has + been + V3 + by + O  

4. Present Perfect Continuous Tense  

Aktif : S + have/has + been + V1-ing + O  

Pasif : S + have/has + been + being + V3 + by + O  

5. Simple Past Tense  

Aktif : S + V2 + O  

Pasif : S + to be (was, were) + V3 + by + O  

6. Past Continuous Tense  

Aktif : S + to be (was, were) + V1-ing + O  

Pasif : S + to be (was, were) + being + V3 + by + O  

7. Past Perfect Tense  

Aktif : S + had not + V3 + O  

Pasif : S + had + been + V3 + by + O  

8. Past Perfect Continuous Tense  

Aktif : S + had + been + V1-ing + O  

Pasif : S + had + been + being + V3 + by + O  

9. Simple Future Tense  

Aktif : S + will not + V1 + O  

Pasif : S + will be + V3 + by + O  

10. Future Continuous Tense  

Aktif : S + will + be + V1-ing + O  

Pasif : S + will + be + being + V3 + by + O  

11. Future Perfect Tense  

Aktif : S + will + have + V3 + O  

Pasif : S + will + have + been + V3 + by + O  

12. Future Perfect Continuous Tense  

Aktif : S + will + have + been + V1-ing + O  

Pasif : S + will + have + been + being + V3 + by + O  

13. Simple Future Past Tense  

Aktif : S + would + V1 + O  

Pasif : S + would + be + V3 + by + O  

14. Future Past Continuous Tense  

Aktif : S + would + be + V1-ing + O  

Pasif : S + would + be + being + V3 + by + O  

15. Future Past Perfect Tense  

Aktif : S + would + have + V3 + O  

Pasif : S + would + have + been + V3 + by + O  

16. Future Past Perfect Continuous Tense  

Aktif : S + would + have + been + V1-ing + O  

Pasif : S + would + have + been +being + V3 + by + O

Answer:

Recount text is a kind of text which is used to retell a past event or story.

Recount text is used to inform or entertain the readers.

The generic structure of recount text

1. Orientation

2. Events

3. Re-orientation

Here is the recount text which tells about watching Danur / Jalangkung 2. I choose Danur for this text.

Watching Danur

Last week, I went to the cinema with my older sister. She invited me to watch one of the most popular Indonesia horror movies entitled Danur.

It was my first time to watch Indonesia horror movie. At first, i didn't want to watch it because it was Indonesia horror movie. To be honest, I didn't like to watch Indonesia horror movies because i thought they weren't scary at all. I decided to watch it after my sister had told me that the rate of the movie was great.

After watching it, i admitted that the movie deserved its rating. It was so great. I didn't imagine that the movie could be so tense and creepy. It made me interested to watch other Indonesia horror movie.
