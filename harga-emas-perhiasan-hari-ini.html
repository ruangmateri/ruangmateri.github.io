Besarnya nilai jasa taksiran dapat dihitung dengan rumus:

Nilai Jasa Taksiran = (Berat Emas x Harga Emas per gram) - (Pinjaman x (1 + Suku Bunga x Jumlah Hari / 360))

Dalam kasus ini:

Berat Emas = 5 gram

Harga Emas per gram = Rp. 700.000 / 5 gram = Rp. 140.000

Pinjaman = Rp. 3.500.000

Suku Bunga = 3% per 15 hari

Jumlah Hari = 20 hari

Substitusikan nilai-nilai tersebut ke dalam rumus:

Nilai Jasa Taksiran = (5 gram x Rp. 140.000) - (Rp. 3.500.000 x (1 + 0,03 x 20 / 360))

Menghitung suku bunga per hari:

Suku Bunga per hari = Suku Bunga x Jumlah Hari / 360 = 0,03 x 20 / 360 = 0,00166667

Menghitung nilai jasa taksiran:

Nilai Jasa Taksiran = (5 gram x Rp. 140.000) - (Rp. 3.500.000 x (1 + 0,00166667))

Nilai Jasa Taksiran = Rp. 700.000 - (Rp. 3.500.000 x 1,00166667)

Nilai Jasa Taksiran = Rp. 700.000 - Rp. 3.503.333,34

Nilai Jasa Taksiran ≈ -Rp. 2.803.333,34

Dalam kasus ini, nilai jasa taksiran akan negatif (-Rp. 2.803.333,34). Hal ini menunjukkan bahwa ibu Lina akan mendapatkan jumlah yang lebih rendah dari nilai pinjaman karena harus membayar bunga dan jasa taksiran.
