Rumus Tripel Pythagoras

Tiga bilangan berikut tripel pythagoras dan bukan puthagoras

a = 9, b = 13, c = 15

a = 7, b = 12, c = 15

   = [tex]\sqrt{7^{2} + 12^{2} }[/tex]

   = [tex]\sqrt{49 + 144 }[/tex]

   = [tex]\sqrt{193}[/tex]

c. 10, 24, 25

a = 10, b = 24, c = 25

Bukti:

c² = 10² + 24²

   = [tex]\sqrt{10^{2} + 24^{2} }[/tex]

   = [tex]\sqrt{100 + 576} }[/tex]

   =[tex]\sqrt{676}[/tex]

Tidak terbukti. Jadi, tiga bilangan tersebut bukan tripel pythagoras.

d. 8, 15, 17

a = 8, b = 15, c = 17

Bukti:

   = [tex]\sqrt{8^{2} + 15^{2} }[/tex]

25. Tuliskan tiga contoh dari Tripel Pythagoras


Berikut adalah 3 contoh Tripel Pythagoras :

3, 4, 55, 12, 137, 24, 25

Pembahasan

Pada segitiga siku-siku berlaku teorema Pythagoras yang menyatakan bahwa kuadrat sisi miring (hipotenusa) merupakan penjumlahan dari kuadrat sisi-sisi lainnya. Misal a dan b adalah sisi siku-siku dan c adalah sisi miring maka dapat dirumuskan :

c² = a² + b²

Tripel Pythagoras adalah pasangan 3 bilangan asli yang memenuhi teorema Pythagoras. Bilangan terbesar merupakan hipotenusa.

Berikut adalah contoh tripel Pythagoras :

3, 4, 55, 12, 137, 24, 25

Pembuktian :

# 3, 4, 5

5² = 3² + 4²

25 = 9 + 16

25 = 25 (terbukti)

# 5, 12, 13

13² = 5² + 12²

169 = 25 + 144

169 = 169 (terbukti)

# 7, 24, 25

25² = 7² + 24²

625 = 49 + 576

625 = 625 (terbukti)
