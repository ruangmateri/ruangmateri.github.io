32. If you got a bunch of money,you...(spend) it on stupid things?


if you got a bunch of money, would you spend it on stupid things?
jawaban ini masuk di conditional sentence type 2.
if+ v2, s+would+ v1.
berhubung itu yang kamu tanyakan bentuk question jadinya
rumusnya if+ v2, would+s+ v1+?If you got a bunch of money, you would spend it on stupid things.

Kalimat ini merupakan contoh kalimat Second Conditional/Conditional Type 2.

menurut ef.co.id, ini adalah rumus Second Conditional:

If + simple past, would/ could/ might + bare infinitive atau would/ could/ might + bare infinitive + simple past.

bentuk terbaliknya:

would/ could/ might + bare infinitive atau would/ could/ might + bare infinitive + simple past + if + simple past.
