1. apa perbedaan sou desu ne dengan sou desu ka ? Perbedaan soudesuka dan soudesuka


そうですね (Sou desu ne) : Iya ya, Benar juga, Iya kan?, dsb.そうですか (Sou desu ka) : Iya kah?, Benar kah?, Oh begitu?, dsb.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kedua ungkapan 'Sou desu ne' dan 'Sou desu ka' sama-sama terdapat 'Sou desu'. Hanya berbeda di akhirnya saja. 'Sou desu ne' menggunakan akhiran partikel Ne, sedangkan 'Sou desu ka' menggunakan akhiran partikel Ka. Sebelum ke pembahasan mengenai arti dari kedua ungkapan ini, akan saya bahas dulu mengenai partikel Ne dan Ka.

Fungsi Partikel ね (Ne)

Partikel ini merupakan salah satu partikel dalam kelompok Shuujoshi (partikel akhir kalimat). Fungsinya adalah untuk menyatakan kesan, serta seperti untuk menanyakan persetujuan lawan bicara. Kalau diartikan kurang lebih seperti "..ya", "..bukan?", "..kan?", dsb.

Cth: Kawaii ne (Imut ya), Kore desu ne (Ini kan?)

Fungsi Partikel か (Ka)

Partikel ini juga termasuk ke dalam kelompok Shuujoshi. Fungsinya adalah untuk membuat kalimat tanya. Kalau diartikan kurang lebih seperti "..kah?", "..nggak?", atau bisa juga hanya berperan seperti tanda tanya.

Cth: Ii desu ka (Boleh?), Kore desu ka (Ini kah?)

- - - - - - -

Setelah memahami kegunaan dari kedua partikel tersebut, mari kita lanjut ke pembahasan utama dari soal ini. Kedua ungkapan 'Sou desu ne' dan 'Sou desu ka' sama-sama terdapat 'Sou desu' di dalamnya.

Arti そうです (Sou desu)

Ungkapan ini bisa diartikan "Ya", "Benar", dsb. Merupakan ungkapan persetujuan terhadap pernyataan dari lawan bicara. Ungkapan ini merupakan ungkapan yang sopan. Untuk membuatnya menjadi ungkapan yang kasual, dapat dengan menghilangkan 'Desu'.

Contoh percakapan satu:

A : Azu-kun desu ka? (Apa kamu Azu?)B : Hai, sou desu (Ya, benar)

Contoh percakapan dua:

A : Kou? (Begini?)B : Sou sou (Ya ya)

Arti そうですね (Sou desu ne)

Ungkapan ini dapat berarti "Iya ya", "Benar juga", "Iya kan?", dsb. Tergantung dari intonasinya, ungkapan ini juga bisa menjadi ungkapan untuk menanyakan persetujuan lawan bicara. Ungkapan ini bahkan juga bisa hanya menjadi ungkapan untuk merespon pendapat lawan bicara, tanpa mengandung makna persetujuan.

Contoh percakapan satu:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ne (Iya ya)

Contoh percakapan dua:

A : Dou omou? (Bagaimana pendapatmu?)B : Sou desu ne... (Hmm...)

Arti そうですか (Sou desu ka)

Ungkapan ini memiliki arti "Iya kah?", "Benar kah?", "Oh begitu?", dsb. Digunakan sebagai ungkapan untuk bertanya tentang kebenaran dari pernyataan lawan bicara. Selain itu juga bisa menjadi ungkapan untuk menyatakan ketidaksetujuan.

Contoh percakapan satu:

A : Ano hito wa Azu-san (Orang itu Azu)B : Sou desu ka (Oh begitu)

Contoh percakapan dua:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ka (Ngga ah)
