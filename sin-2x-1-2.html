sin 2x = sin 120° untuk 0° ≤ x ≤ 180° adalah {30°, 60°}. Nilai tersebut diperoleh dari perhitungan dengan rumus penyelesaian persamaan trigonmetri. Simak pembahasan berikut.

Pembahasan

Diketahui:

sin 2x = sin 120° untuk 0° ≤ x ≤ 180°

Ditanya: Himpunan penyelesaian untuk x

Jawab:

Penyelesaian persamaan trigonometri untuk sinus

sin x = sin α

Maka

x = α + k × 360°

x = (180° - α) + k × 360°

dengan k = 0, 1, 2, 3, 4, ...

Dari soal diketahui sin 2x = sin 120° untuk 0° ≤ x ≤ 180°, maka diperoleh

sin 2x = sin 120° untuk 0° ≤ x ≤ 180°

cara 1:

sin 2x = sin 120°

2x = 120° + k × 360°

x = 120°/2 + k × 360°/2

x = 60° + k × 180°

untuk k = 0

x = 60° + k × 180°

x = 60° + 0 × 180°

x = 60° + 0°

x = 60°

untuk k = 1

x = 60° + k × 180°

x = 60° + 1 × 180°

x = 60° + 180°

x = 240° (tidak memenuhi)

cara 2:

sin 2x = sin 120°

2x = (180° - 120°) + k × 360°

2x = 60° + k × 360°

x = 60°/2 + k × 360°/2

x = 30° + k × 180°

untuk k = 0

x = 30° + k × 180°

x = 30° + 0 × 180°

x = 30° + 0°

x = 30°

untuk k = 1

x = 30° + k × 180°

x = 30° + 1 × 180°

x = 30° + 180°

x = 210° (tidak memenuhi)

Jadi himpunan penyelesaian dari x adalah {30°, 60°}

Trigonometri adalah sebuah metode matematika yang membandingkan antara sisi dan sudut. Trigonometri digunakan dalam matematika untuk mencari panjang sisi lain dengan membandingkan sisi dengan sudut yang telah diketahui.

Trigonometri ini sendiri terbagi menjadi 6, diantaranya sinus (sin), cosinus (cos), tangen (tan), cosecan (csc), secotan (sec), cotangen (cot).

Sin(x) = Sin(270°)

Kita tahu bahwa sinus 270° sama dengan -1, jadi persamaan ini dapat diselesaikan dengan mengganti nilai sinus x dengan -1:

Sin(x) = -1

x = 3π/2

Sin(3x) = Sin(210°)

Pertama, kita perlu mencari sudut-sudut di dalam interval 0≤x≤2π yang memiliki sinus 210°. Sudut yang memiliki sinus 210° adalah 30° dan 150°. Karena sinus adalah fungsi periodik, kita dapat menuliskan:

3x = 30° + 2πk atau 3x = 150° + 2πk

x = 10° + (2π/3)k atau x = 50° + (2π/3)k
