Turunan adalah kecepatan perubahan nilai output fungsi relatif terhadap nilai input fungsi dimana perubahan nilai input mendekati 0

Untuk f(x) = y, dan f(x + x1) = y1, maka (y1-y)/(x1) = turunan

turunannya adalah (12x³ + 4x)

turunan dari turunnnya adalah (36x² +4)
