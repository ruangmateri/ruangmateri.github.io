Jadi model matematikanya adalah:

Maksimumkan: f(x, y) = 150.000x + 100.000y

Dengan kendala: x + y ≤ 25; 2x + 3y ≤ 60; x ≥ 0; y ≥ 0

Harga tiket sebuah wahana yaitu tiket dewasa 60.000. Tiket anak-anak 35.000. Satu gerombolan terdiri 50 orang membeli x tiket dewasa dan y tiket anak-anak. Jika uang pembeliannya 2.125.000. Model matematika dari permasalahan tersebut adalah

x + y = 50

60.000x + 35000y = 2.125.000

dengan penyelesaian :

Anak-anak ada sebanyak 35 orang

Dewasa ada sebanyak 15 orang

PEMBAHASAN

Persoalan ini adalah menyangkut tentang Sistem Persamaan Linier Dua Variabel yang bisa di selesaikan dengan salah satu dari metode berikut:

1. Metode Substitusi

2. Metode Eliminasi

3. Metode Grafik

Marilah kita mula-mula terjemahkan soal yang dimaksud dalam persamaan matematika berikut.

Satu gerombolan terdiri 50 orang membeli x tiket dewasa dan y tiket anak-anak , maka bisa di tulis :

x + y = 50

x = 50 - y → Persamaan 1

Jika uang pembeliannya 2.125.000 dengan harga tiket dewasa 60.000 dan tiket anak-anak 35.000 , maka bisa di tuliskan :

60.000x + 35000y = 2.125.000

12x + 7y = 425 ( sama-sama di bagi 5000 )

12(50 -y) + 7y = 425  ← Persamaan 1

600 - 12y + 7y = 425

-5y = 425 - 600

-5y = -175

y = 175/ 5

y = 35

x = 50 - y = 50 - 35 = 15

Kesimpulan:

Anak-anak ada sebanyak 35 orang

Dewasa ada sebanyak 15 orang

Model matematikanya

x adalah karcis anak, y adalah karcis dewasa

200x + 300y = 42.000

x + y = 180

Nilai x dan y = ?

200x + 300y = 42.000 ... 1

x + y = 180 ... 2

-- Persamaan 2 dikalikan 200 --

200x + 300y = 42.000

200x + 200y = 36.000 ... 3

-- Metode Eliminasi, persaman 1 dikurang dengan persamaan 3 --

100y = 6.000

x + y = 180

Berdasarkan informasi yang disampaikan dalam pertanyaan, kita dapat menentukan model matematika untuk menyelesaikan masalah ini sebagai berikut:

Jumlah ongkos penumpang dewasa = jumlah penumpang dewasa x Rp 5.000,00 = n x 5.000

Jumlah ongkos penumpang anak-anak = jumlah penumpang anak-anak x Rp 3.500,00 = m x 3.500

Jumlah ongkos semua penumpang = jumlah ongkos penumpang dewasa + jumlah ongkos penumpang anak-anak = (n x 5.000) + (m x 3.500)

Jumlah ongkos semua penumpang = Rp 180.000,00 = (n x 5.000) + (m x 3.500)

Model matematika yang dapat digunakan untuk menyelesaikan masalah ini adalah sebagai berikut:

Jumlah ongkos semua penumpang = Rp 180.000,00 = (n x 5.000) + (m x 3.500)

Untuk menentukan jumlah penumpang dewasa dan anak-anak, kita dapat menggunakan model matematika di atas. Namun, kita perlu melakukan beberapa perhitungan tambahan untuk menemukan jumlah penumpang dewasa dan anak-anak tersebut.

Contoh:

Jika jumlah penumpang dewasa adalah 20 orang, maka jumlah ongkos penumpang dewasa adalah 20 x Rp 5.000,00 = Rp 100.000,00

Jika jumlah penumpang anak-anak adalah 25 orang, maka jumlah ongkos penumpang anak-anak adalah 25 x Rp 3.500,00 = Rp 87.500,00

Jumlah ongkos semua penumpang adalah Rp 100.000,00 + Rp 87.500,00 = Rp 187.500,00

Dari contoh di atas, jumlah penumpang dewasa dan anak-anak adalah 20 orang dan 25 orang, sehingga kita dapat menyimpulkan bahwa jumlah penumpang dewasa dan anak-anak adalah 20 orang dan 25 orang.
