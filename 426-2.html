24. tentukan konversi bilangan 426(10)=...(8) dan 1010.011(2)=...(10)


Materi : Sistem Komputer
Kategori : Konversi bilangan

Langkah Penyelesaian dan Jawaban :

1. Ubahlah bilangan desimal berikut ini ke bilangan oktal :
426₍₁₀₎ = .... ₍₈₎

Gunakan modulus/sisa hasil bagi untuk mengetahui konversinya :
426 / 8 = 53, sisa 2
53 / 8 = 6, sisa 5
6/8 = 0, sisa 6

Susunlah dari bawah ke atas, maka hasilnya adalah :
426₍₁₀₎ = 652₍₈₎

2. Ubahlah bilangan biner berikut ini ke bilangan desimal :
1010.011₍₂₎ = .... ₍₁₀₎

- Bagi menjadi 2 bagian dahulu yaitu 1010 dan 011, maka :
1010 = (1 × 2³) + (0 x 2²) + (1 x 2¹) + (0 × 2⁰)
1010 = 8 + 0 + 2 + 0
1010 = 10

- Untuk bagian ke 2 (011) karena dibelakang koma, maka :
011 = (0 × 2⁻¹) + (1 × 2⁻²) + (1 x 2⁻³)
011 = 0 + 1/4 + 1/8
011 = 3/8
011 = 0,375

Jumlahkan :
1010.011 = 10 + 0,375
1010.011 = 10,375

Maka hasilnya adalah :
1010.011₍₂₎ = 10,375₍₁₀₎
