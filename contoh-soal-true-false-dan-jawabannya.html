4. Peter bought the cellphone one month ago.

➡️ Peter membeli ponsel satu bulan yang lalu.

Berdasarkan kalimat alinea kedua "Yesterday, I purchased a cellphone .. "Yang artinya "Kemarin, saya beli sebuah ponsel .. "

Jadi, jawaban FALSE (F) karena yang benar belinya kemarin.

5. There is a defect in the call quality.

➡️ Ada cacat pada kualitas panggilan.

Berdasarkan kalimat "But the main purpose of a phone is to make calls and this one doesn't seem good for that."Yang artinya "Tapi tujuan utama telepon adalah untuk melakukan panggilan dan yang ini sepertinya tidak bagus untuk itu."

Jadi, jawaban TRUE (T).

6. Peter think that the best feature in the cellphone is a camera.

➡️ Peter berpikir bahwa fitur terbaik di ponsel adalah kamera.

Berdasarkan kalimat dalam paragraf kedua "I am impressed with the quality of camera and other features."Yang artinya "Saya terkesan dengan kualitas kamera dan fitur lainnya."

Jadi, jawaban FALSE (F) karena yang benar Peter Parker hanya terkesan dengan fitur kamera dan lainnya.

7. Peter bought the cellphone from the Alit Inc. for the first time.

➡️ Peter membeli ponsel dari Alit Inc. untuk pertama kalinya.

Berdasarkan kalimat di awal alinea ketiga "This is not the first time I purchased phones built by your company .. "Yang artinya "ini bukan pertama kalinya saya beli ponsel dari perusahaan kalian .. "

Jadi, jawaban FALSE (F) karena yang benar bukan pertama kalinya.

8. Peter is dissatisfied with Alit Inc.

➡️ Peter tidak puas dengan Alit Inc.

Teks di atas adalah "complaint letter"Yang artinya "surat keluhan"

Jadi, jawaban TRUE (T) karena kalo puas buat apa bikin surat keluhan.

10. Kode berikut ini yang akan menghasilkan nilai true adalah....a. if (true != false && false == false) b. if (false != false II true != true)c. if (true == true && false != false)d. if (false == false && true == false)​


Jawaban:

Jawaban yang benar adalah:

c. if (true == true && false != false)

Penjelasan:

Penjelasan:

Pada a, kondisi pertama (true != false) benar, namun kondisi kedua (false == false) juga benar. Karena ada operator AND (&&) maka hasilnya false.

Pada b, kondisi pertama dan kedua sama-sama salah (false != false dan true != true). Karena ada operator OR (||) maka hasilnya juga false.

Pada c, kondisi pertama (true == true) benar dan kondisi kedua (false != false) juga benar. Karena ada operator AND (&&) maka hasilnya true.

Pada d, kondisi pertama (false == false) benar, namun kondisi kedua (true == false) salah. Karena ada operator AND (&&) maka hasilnya false.

22. 11.(P AND NOT(Q OR NOT R)) hanya bernilai TRUE apabila P,Q,dan R berturut-turut bernilaiA. TRUE, TRUE, TRUEB. TRUE, FALSE, TRUEC. TRUE, FALSE, FALSED. TRUE, TRUE, FALSEE. FALSE, TRUE, TRUE ​


Jawaban:

D. TRUE, TRUE, FALSE

Penjelasan:

Untuk menyelesaikan masalah ini, kita perlu menerapkan aturan logika Boolean, di mana NOT mengambil prioritas tertinggi, kemudian OR, dan terakhir AND. Dengan demikian, kita dapat menyelesaikan masalah ini dengan membuat tabel kebenaran yang menggambarkan nilai P, Q, R, dan hasil ekspresi P AND NOT(Q OR NOT R). Dari tabel di atas, kita dapat melihat bahwa P AND NOT(Q OR NOT R) hanya bernilai TRUE jika P benar (TRUE), Q salah (FALSE), dan R benar (TRUE). Oleh karena itu, jawaban yang benar adalah D.
