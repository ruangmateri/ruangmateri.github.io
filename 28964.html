28. 8, 27, 64, 125, 216,...A. 324B. 256C. 289D. 343​


~Barisan dan Deret

___________________

✧ Penyelesaian

• Menentukan pola barisan

Perhatikan jika pola dijabarkan akan menjadi:

U1 = 8 → (1 + 1)³ ➞ 2³U2 = 27 → (2 + 1)³ ➞ 3³U3 = 64 → (3 + 1)³ ➞ 4³U4 = 125 → (4 + 1)³ ➞ 5³U5 = 216 → (5 + 1)³ ➞ 6³dan seterusnya ...

Karena pola pada barisan tersebut sudah diketahui, didapat pola Un = (n + 1)³

...

• Menentukan suku ke-6 (U6)

Un = (1 + n)³

U6 = (6 + 1)³

U6 = 7³

U6 = 7 × 7 × 7

U6 = 343

» Kesimpulan

Jadi, Suku berikutnya dari barisan 8, 27, 64, 125, 216 ... adalah 343 → Opsi D
