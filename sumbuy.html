29. 11.Persamaan grafik fungsi kuadrat yang melalui sumbuY dititik (0,3) dan mempunyai titik puncak (1,4) adalah….​


Persamaan grafik fungsi kuadrat yang melalui sumbuY dititik (0,3) dan mempunyai titik puncak (1,4) adalah y = -x² + 2x + 3

PEMBAHASAN

Fungsi kuadrat memiliki bentuk umum y = ax² + bx + c

dimana :

x = -b/2a → persamaan sumbu simetri

y = ( b² - 4ac ) / (-4a) → nilai maksimum fungsi

Untuk mencari persamaan grafik fungsi kuadrat maka :

Skenario 1 :

Jika diketahui titik potong dengan sumbu X yakni di x₁ dan x₂ maka:

y = a ( x - x₁ ) ( x - x₂ )

Skenario 2 :

Jika diketahui titik puncak kurva ( p , q ) maka :

y = a(x - p)² + q

Skenario 3 :

Jika diketahui 3 titik sembarang pada kurva maka :

y = ax² + bx + c  

Okay dengan prinsip ini kita akan mencoba selesaikan soalnya.

Berhubung diketahui titik puncak (1,4) dari grafik fungsi kuadrat maka kita akan gunakan Skenario 2 , yakni :

y = a(x - p)² + q

y = a(x - 1)² + 4

Selanjutnya untuk mencari nilai a kita bisa masukkan titik potong sumbu Y di ( 0, 3 ) sehingga :

y = a(x - 1)² + 4

3 = a(0 - 1)² + 4

3 = a(1) + 4

a = 3 - 4

a = -1

Persamaan kuadrat yang dimaksud adalah:

y = -1(x-1)² + 4

y = - (x²-2x+1) + 4

y = -x² + 2x + 3
