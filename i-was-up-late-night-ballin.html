11. apa bahasa indonesianya i usually get up late​


Jawaban:

saya biasanya bangun terlambat

Vocabulary (kosa kata)

I usually get up late:

I = aku

usually = biasanya

get up = bangun (dari tidur)

late = terlambat

.

I usually get up late = saya biasanya bangun terlambat

.

kalimat ini merupakan kalimat verbal dalam simple present tense karena kebiasan ditandai dengan USUALLY. ingat rumus present tense:

I/you/they/we + verb1

she/he/it + verb1 +s/es

contohnya:

I usually wake up early = saya biasanya bangun awal

she always goes to school = dia selalu pergi ke sekolah
