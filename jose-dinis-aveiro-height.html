11. 5/6 of Wendy's height is equal to 2/3 of her mother's height the difference in their height is 3 /10 m what is Wendy's height express your answer in its simplest form​


Wendy's height is [tex]1\frac{1}{5} metres[/tex]

--------------------------

Let's say Wendy's height is [tex]x[/tex] and her mother's height is [tex]y[/tex]

We know that

[tex]\frac{5}{6} x = \frac{2}{3} y[/tex]

that means

[tex]x=\frac{2}{3} y\div \frac{5}{6} \rightarrow x=\frac{4}{5}y[/tex]

We also know that [tex]y-x = \frac{3}{10} m[/tex]

Using the substitution method, we get

[tex]y-\frac{4}{5} y = \frac{3}{10} m\\\\\frac{5}{5}y- \frac{4}{5} y = \frac{3}{10} m\\\\\frac{1}{5} y = \frac{3}{10} m\\\\y=\frac{3}{10} m\div\frac{1}{5} \\y=\frac{3}{10} m\times\frac{5}{1} \\\\y=\frac{15}{10}m\\\\[/tex]

After we found the value of [tex]y[/tex], we return to [tex]y-x = \frac{3}{10} m[/tex] to find the value of [tex]x[/tex].

[tex]y-x = \frac{3}{10} m[/tex]

[tex]\frac{15}{10}m -x = \frac{3}{10} m\\x=\frac{15}{10}m -\frac{3}{10} m\\\\x=\frac{12}{10} m =1 \frac{1}{5} m\\\\\therefore Wendy's\ height\ is\ 1 \frac{1}{5} m[/tex]

t = s + 26

9s = 7t

9s = 7(s + 26)

9s = 7s + 182

2s = 182

s = 91

So

s + t

= s + s + 26

= 2s + 26

= 182 + 26

= 208 cm

¾x = ⅖x + ¹⁵⁴/5

¹⁵/20x - ⁸/20x = ¹⁵⁴/5

⁷/20x = ¹⁵⁴/5

x = ¹⁵⁴/5 ÷ ⁷/20

x = ¹⁵⁴/5 x 20/7

x = ⁴⁴⁰/5

x = 88cm

If + simple past tense, simple past future/past modal

(+) S + Verb 2 + O(–) S + didn't + Verb 1 + O(?) Did + S + Verb 1 + O

Simple Past Future

Simple Past Modal

(+) S + could/might + Verb 1 + O(–) S + couldn't/might not+ Verb 1 + O(?) Could/Might + S + Verb 1 + O
