18. 4. My brother would drive to school in his carif there ........... (be) enough gas in it.​


PART = CONDITIONAL SENTENCE

-----------------------

Jawaban:

My brother would drive to school in his car if there were enough gas in it.

NOTE :

Struktur IF conditional menggunakan type ke 2 , bisa dicermati pada result clause nya "would drive " , maka untuk verb pada IF clause menggunakan bentuk PAST . BE yang sesuai adalah menggunakan WERE karena keformalitasan penulisan baik subyek tunggal maupun jamak. Penggunaan WAS dianggap tidak formal dalam penulisan IF conditional dan hanya dipakai pada percakapan keseharian.

Penjelasan:

CONDITIONAL SENTENCE/ ANDAI-ANDAI       tells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition.       Consists -If clause and result clause.       Using Comma when -IF result clause is started at the beginning sentence. Otherwise, there is no comma when -IF clause is put after the result clause.      

There are 4 types of Conditional Sentence      

1. CONDITIONAL ZERO      

is used to describe the PHENOMENON and GENERAL TRUTH ( THE FACT )      

Pattern :      

IF+S+base verb (s/es ) + O/C , S+base verb (s/es ) + O/C      

Example :      

If it rains, the streets get wet.      

Fact : it's the general truth or phenomenon.

2. CONDITIONAL TYPE 1      

is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened )       In fact , the speaker will actualize his/her dream now or soon.      

Pattern :      

If +S+ verb (s/es) + O, S+will+verb1+O/C      

Example :      

If i have some money, i will buy the English grammar book later.      

Fact : it's possible that i will buy the English grammar book today or later because i have enough money.

3. CONDITIONAL TYPE 2      

is used to describe  the untruth in the present / future       Contrary to the fact in the present or it could be in the future       The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future       "Keep in mind, if we use the first , second person and third person singular  ( I, she, he, it , ) , it must be followed by the PAST linking verb "were."" It is the same way to the plural form as the formal writing." If we use the PAST linking verb "was " , it should be used in the colloquial as the informal conversation.      

Pattern:      

If+S+verb2+O/C , S+Would+verb1+O/C      

Example :      

If i were a president, i would build the largest library in the southeast Asia.

Fact in the present/future : I am not a president but i'm a dentist now.      

     

4. CONDITIONAL TYPE 3      

Is used to describe  the untruth in the past       Contrary to the fact in the past      

Pattern :      

If+S+Had+V3+O/C  ,S+Would+have+V3+O/C      

Example :      

If i hadn't lost my money , i would have bought the new cooking recipe.      

Fact in the past  : i lost my money, so i didn't buy the new cooking recipe yesterday      

     

KESIMPULAN :      

Conditional sentence sebagai kalimat andai-andai atau hanya impian jika persyaratan pendukung nya tidak tercapai.       Conditional Zero membahas tentang fakta/kejadian yang sebenarnya/fenomena alam. Bentuk strukturnya pada umumnya menggunakan simple present.Conditional type 1 membahas tentang prediksi/rencana nyata yang mungkin saja terwujud karena persyaratannya mampu untuk direalisasikan saat ini atau nanti-nanti.       Conditional type 2 membahas tentang kejadian yang kontras ( berkebalikan ) dengan fakta nya yang terjadi saat ini atau mendatang. Dalam istilah ini, si pengucap hanya berandai-andai saja ( Memiliki niat tetapi tidak mampu untuk mewujudkannya) karena persyaratannya tidak terpenuhi untuk saat ini atau nanti-nanti .       Conditional type 3 membahas tentang kejadian yang kontras ( berkebalikan ) dengan faktanya yang terjadi di waktu lampau. Si pengucap mengulas kembali kejadian yang tidak berhasil diwujudkan di waktu lampau. Ini seolah-olah, ada pernyataan penyesalan yang diliput kembali. Jika dalam IF conditional type 2 terdapat BE maka , baik subyek plural maupun tunggal harus menggunakan were sebagai bentuk formal dalam penulisan. Sedangkan was dianggap tidak formal , hanya digunakan pada percakapan sehari-hari.
