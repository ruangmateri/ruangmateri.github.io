Préface

Pronom COD (complément d’objet direct) adalah pronomina yang digunakan untuk menggantikan obyek langsung dari suatu kalimat. Obyek tersebut tidak boleh didahului oleh preposisi (di, ke, untuk, dll).

Par exemples ...

Je lis le livre. / Saya membaca buku.

→ Je le lit. / Saya membacanya.

Tu achètes la voiture. / Kamu membeli mobil.

→ Tu l'achètes. / Kamu membelinya.

Nous aimons les chats. / Kami menyukai kucing-kucing.

→ Nous les aimons. / Kami menyukai mereka.

Les pronoms COD dalam bahasa Prancis yaitu:

me/m' → saya (me)te/t' → kamu (you)le/la/l' → dia/-nya (him/her/it)nous → kami/kita (us)vous → Anda/kalian (you)les → mereka (them)

Umumnya pronom COD diletakkan sebelum kata kerja.

Pronom COD juga digunakan untuk menggantikan ungkapan yang menggunakan verba.

Par exemple ...

Je lui demande de partir. / Saya meminta dia untuk pergi.

→ Je lui le demande.

Susunlah kata kata di bawah ini menjadi kalimat kalimat yang benar 21. mangent / leur / au / professeur / Silvie et Jeanne/ restaurant / avec 22. maison / la / regardes / à / télévision / tu / la 23. classe / sont / les / la / dans / étudiants 24. avons / quatre / Bandung / à / nous / maisons 25. de / c’est / ma / la / mère / chambre


Le structure de la phrase

21. Silvie et Jeanne mangent avec leur professeur au restaurant.

22. Tu regardes la télévision à la maison.

23. Les étudiants sont dans la classe.

24. Nous avons quatre maisons à Bandung.

25. C'est la chambre de ma mère.

-----------------

L'explication

Sebuah kalimat declarative atau pernyataan dalam bahasa Perancis tersusun dari subjek (GNS) + kata kerja (V).

Contoh kalimat sederhana:

Saya membaca.

Je lire.

GNS + V

Bila ada objek langsung (COD), letakkan setelah kata kerja.

Saya membaca buku.

Je lire le livre.

GNS + V + COD

Jika objek langsung berupa kata ganti, letakkan sebelum kata kerja.

Saya membacanya.

Je le lire.

GNS + pronom COD + V

Keterangan atau compléments circonstanciels dapat diletakkan di depan, di akhir, atau di awal kalimat.

Saya membaca buku sekarang.

Je lire le livre maintenant.

GNS + V + COD + CCT

Di perpustakaan, saya membaca buku.

À la bibliothèque, je lire le livre.

CCL + GNS + V + COD

Saya membaca buku dengan pelan.

Je lire lentement le livre.

GNS + V + CCM + COD  

*CCT = complément circonstanciel de temps (keterangan waktu)

*CCL = complément circonstanciel de lieu (keterangan tempat)

*CCM = complément circonstanciel de manière (keterangan cara)

Le complément de détermination du nom atau CDN adalah perluasan dari sebuah kata benda. CDN memberi keterangan lebih yang membuat sebuah kata benda menjadi lebih spesifik. CDN berupa kata/frasa benda yang terkoneksi pada kata benda yang dijelaskannya menggunakan kata depan de, à, en.

Contoh:

kentang = la pomme de terre (apel dari tanah)skateboard = le planche à roulettes (papan di roda)piring porselen = l'assiette en porcelaine (piring dari porselen)

C'est = ini (adalah)

C'est diikuti determiner lalu kata benda.

Contoh:

Ini (adalah) sebuah pensil.

C'est un crayon.

Ini (adalah) pensil saya.

C'est mon crayon.

Ini (adalah) pensil Alex.

C'est le crayon de Alex.
