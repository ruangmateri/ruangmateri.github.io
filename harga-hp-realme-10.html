10. Halo, Maaf Izin Bertanya..Kalau Di Kalkulator Realme Kenapa Hasil 5.000 × 20.000 itu 1,E8? itu memang seperti itu? tolong di jelaskan.. terimakasih l​


5.000×20.000=1E8

1E8=100.000.000

1E8 maksudnya angka 0-nya ada 8 digit

5.000 angka 0-nya ada 3 digit

20.000 angka 0-nya ada 4 digit

2×5=10

10 angka 0-nya ada 1 digit

3+4+1=8

Jadi, angka 0-nya digabungkan jadi ada 8 digit.

5.000 x 20.000 itu hasilnya 100.000.000

Maksud dari "1,E8" adalah "1 . Eksponen 8"

Eksponen 8 itu maksudnya 10^8

Karena, 100.000.000 = 10^8

Penulisan "1,E8" digunakan untuk menghemat ruang kalkulator saat menampilkan hasil operasi hitung.

Notasi Ilmiah/bentuk baku merupakan cara menyatakan angka-angka yang terlalu besar atau terlalu kecil dalam bentuk a x 10ⁿ di mana -10 < a < 10 dan n = bilangan bulat.

Bilangan lebih besar atau sama dengan 10, gunakan sebuah pangkat positif ketika memindahkan titik desimal ke kiri.Bilangan antara 0 dan 1, gunakan sebuah pangkat negatif ketika memindahkan titik desimal ke kanan.

Penjelasan Soal:

0,38 0,000546 0,0505050,000009

Ditanya:

Bentuk baku

Jawab:

0,38 bilangan antara 0 dan 1, gunakan sebuah pangkat negatif ketika memindahkan titik desimal ke kanan.

Banyak pergeseran ke kanan adalah 1, sehingan n = -1

a = 3,8

n = -1

Notasi ilmiah → 0,38 = 3,8 x 10⁻¹

0,000546

Banyak pergeseran ke kanan adalah 4, sehingan n = -4

a = 5,46

Notasi ilmiah → 0,000546 = 5,46 x 10⁻⁴

0,050505

Banyak pergeseran ke kanan adalah 2, sehingan n = -2

a = 5,0505

n = -2

Notasi ilmiah → 0,050505 =  5,0505 x 10⁻²

0,000009

Banyak pergeseran ke kanan adalah 6, sehingan n = -6

a = 9

n = -6

Notasi ilmiah → 0,000009 = 9 x 10⁻⁶

K = 2πr

62,8 = 2 × 3,14 r

62,8 = 62,8 r

r = 10 cm

Volume kerucut = 1/3πr²t

= 1/3 × 3,14 × 10² × 21

= 6594/3

= 2198 cm³

----

bangun ruang biasanya meliputi kubus, balok, prisma, limas, ataupun tabung. Masing-masing dari bangun ruang tersebut memiliki rumus tersendiri untuk menghitung luas ataupun volume dari bangun ruan ruang itu.

sifat sifat kerucut :memiliki sebuah rusuk lengkung memiliki 1 titik puncakmemiliki alas berbentuk lingkaran

rumus volume kerucut :

[tex]{\boxed{\bf \: v = \frac{1}{3} \times \pi \times {r}^{2} \times t }}[/tex]

keterangan :

π = 22/7 atau 3,14r = jari - jarit = tinggi kerucut

[tex]{\boxed{\bf lp = luas \: alas + luas \: selimut }}[/tex]

[tex]{\boxed{\bf lp = (\pi \times {r}^{2}) + (\pi \times r \times s) = \pi \times r(r + s) }}[/tex]

dengan s adalah panjang garis pelukis,yaitu

[tex]{\boxed{\bf s = \sqrt{ {r}^{2} + {t}^{2} } }}[/tex]
