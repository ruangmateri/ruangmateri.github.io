Materi : Persamaan Reaksi
Kata kunci : reaksi setara

a Fe2O3 + b CO ----> c Fe + d CO2

Fe: 2a = c
O: 3a + b = 2d
C: b = d

Substitusikan persamaan2 tsb sehingga didapatkan:

a = 1
b = 3
c = 2
d = 3

Fe2O3 + 3 CO ----> 2 Fe + 3 CO2
