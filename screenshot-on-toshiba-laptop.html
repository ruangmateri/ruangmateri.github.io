23. Bagaimana cara check laptop sudah on berapa lama dari baru?


Pada Windows 8

Buka Windows task manager dengan mengklik kanan pada taskbar lalu pilih Task manager, atau anda juga bisa membukanya dengan menekan tombol CTRL + SHIFT + ESC.

Setelah task manager pilih tab Performance, lalu anda bisa melihat pada Up time, dan itu adalah waktu lamanya komputer anda sudah menyala.

Pada Windows 7/XP/Vista

Buka Windows task manager dengan mengklik kanan pada taskbar lalu pilih Start Task manager, atau anda juga bisa membukanya dengan menekan tombol CTRL + SHIFT + ESC.

referensi gambar: http://indradp-share.blogspot.com/2014/10/cara-mengetahui-waktu-lama-komputer-menyala.html

Lalu pindah ke tab Performance, lalu lihat pada Up time dan itulah berapa lama waktu komputer anda menyala.

referensi gambar: http://indradp-share.blogspot.com/2014/10/cara-mengetahui-waktu-lama-komputer-menyala.html

Sudah tahu kan bagaimana caranya untuk mengetahui berapa lama waktu komputer menyala?, jagalah komputer anda sebaik-baiknya jangan menggunakannya di luar batas kewajarannya, dan jangan sampai 24 jam full, maka komputer anda akan sering ngehang.
