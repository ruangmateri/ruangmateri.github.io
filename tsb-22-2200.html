24. luas selimut kerucut 2200 cm2 ,panjang jari jarinya 14 cm .jika n = 22/7 ,maka panjang garis pelukisnya adalah..........dan tingginya adalah......​


Jawab:

Penjelasan dengan langkah-langkah:

Luas selimut kerucut = π x r x s

r = jari-jari

s = garis pelukis

2200 = (22/7) x 14 x s

2200 = 44 x s

s = 2200/44 = 50

r² + t² = s²

14² + t² = 50²

196 + t² = 2500

t² = 2500 - 196 = 2304

t = √2304 = 48

panjang garis pelukisnya adalah 50 cm

tingginya adalah 48
