19. Untuk menyimpan dalam GitHub Anda harus a. menghubungkan dengan akun GitHub b. menggunakan akun lain C. menambahkan akun lagi d. menggunakan save as e. menyimpan kode​


Jawaban:

Untuk menyimpan dalam GitHub, kamu perlu melakukan langkah-langkah berikut:

a. Menghubungkan dengan akun GitHub: Pertama, kamu perlu membuat akun GitHub jika belum punya. Setelah itu, kamu harus menghubungkan akun GitHub tersebut dengan akun Git di komputermu. Hal ini bisa dilakukan dengan mengatur konfigurasi pada Git, seperti menambahkan alamat email dan nama pengguna GitHub ke dalam Git.

Setelah terhubung, kamu bisa membuat repositori baru di GitHub atau menggunakan repositori yang sudah ada.

b. Menambahkan akun lagi: Tidak perlu menambahkan akun lain untuk menyimpan di GitHub. Yang perlu kamu lakukan adalah menggunakan akun GitHub yang sudah terhubung dengan Git di komputermu.

c. Menggunakan save as: Save as tidak terkait dengan menyimpan kode ke GitHub. Save as digunakan untuk menyimpan file dengan nama baru atau di lokasi yang berbeda di komputermu, bukan untuk menyimpan ke repositori GitHub.

e. Menyimpan kode: Setelah terhubung dengan akun GitHub, kamu bisa menggunakan perintah-perintah Git seperti "git add", "git commit", dan "git push" untuk menyimpan kode ke repositori GitHub. Dengan melakukan commit dan push, perubahan kode yang kamu lakukan akan tersimpan di repositori GitHub.

Jadi, jawabannya adalah A. menghubungkan dengan akun GitHub.

30. definisi bintang termasif


Jawaban:

Bintang yang memiliki massa yang sangat besar, lebih besar dari UY Scuti.

Penjelasan:

Bintang Termasif: RMC 136a1

Apa bedanya bintang terbesar dengan termasif? Besar belum tentu masif. Masif adalah seberapa padat atau berat sebuah objek. Dengan begitu, bintang ini memiliki massa yang sangat besar, lebih besar dari UY Scuti.

Terletak di R163, wilayah terdalam dari kelompok gugus bintang terbuka NGC 2070 (yang berada di galaksi satelit Awan Magellan Besar), bintang ini berada pada jarak sekitar 163.000 tahun cahaya jauhnya dari Bumi di arah rasi bintang Dorado.

RMC 136a1 memiliki massa yang besar, tetapi usianya baru sekitar 800.000 tahun, sehingga ia masih berada pada fase deret utama yang aktif menggabungkan hidrogen ke dalam helium dalam intinya. Bintang ini mengalami kehilangan massa yang ekstrem melalui angin bintangnya yang energik, yang menghembuskan material dari atmosfernya pada kecepatan sekitar 2.600 km per detik.

Pada tahap evolusi saat ini, R136a1 kehilangan massa pada tingkat 5,1 × 10−5 M☉ (3,21 × 10^18 kg per detik) per tahun, yang dengan kata lain kehilangan massanya setara sekitar satu miliar kali lebih tinggi daripada Matahari kehilangan massanya.


31. istilah pada github yang bisa digunakan untuk mengambil repostory dan remote adalah​


Jawaban:

JANGAN LUPA BINTANG 5 NYA YA KAK ^^

Penjelasan:

Istilah pada GitHub yang bisa digunakan untuk mengambil repository dan remote adalah:

1. Clone

Clone digunakan untuk mengambil (copy) repository Git menjadi local repository di komputer Anda. Caranya dengan mengetikkan perintah berikut:

`git clone url`

Dimana `url` adalah alamat URL lengkap repository Git yang ingin Anda clone.

Contoh:

`git clone https://github.com/namlnt/newrepo.git`

Git akan membuat folder `newrepo` lokal dan mengambil semua file dan versi yang ada di repository tersebut.

2. Fork

Fork digunakan untuk membuat salinan (copy) repository milik orang lain ke akun GitHub Anda. Caranya dengan menekan tombol "Fork" pada pojok kanan atas repository di GitHub.

Ini berguna jika Anda ingin mengontribusikan perubahan pada repository milik orang lain.

3. Remote

Git menggunakan istilah remote untuk merujuk ke repository Git yang disinkronkan dengan local repository Anda. Remote bisa berupa repository GitHub atau server Git lainnya.

Remote bisa ditambahkan menggunakan perintah `git remote add` dan disinkronkan dengan perintah `git push` dan `git pull`.

Semoga membantu! Jika ada istilah GitHub lainnya yang ingin Anda ketahui, silakan tanyakan. Saya akan senang membantu selengkap mungkin.

33. infrastruktur masif adalah


Infrastruktur masif adalah masif dalam artian umum adalah sesuatu yang besar dan terjadi secara berkelanjutan. Maka infrastruktur masih adalah suatu infrastruktur yang dibangun dengan skala besar dan secara berkelanjutan.

Jadi Infrastuktur masif dapat diartikan sebagai sarana yang digunakan secara umum untuk kepentingan luas yang dibangun secara terus menerus dan dalam skala yang besar. Dengan adanya infrastruktur masif ini diharapkan perkembangan teknologi tidak hanya ada di wilayah tertentu saja, namun bisa tersebar secara merata ke seluruh pelosok negeri.
