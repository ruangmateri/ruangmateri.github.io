28. 12. Farah : I think we should turn left.Sheila : ... I think we should just go straight.Farah : Well, I'm not sure. I'm afraid we are lost.A. You're right.B. I don't think so.C. I agree with you.D. Let's just turn left.​


B. I don't think so

Terjemahan:

Farah: Aku pikir kita harus belok kiri.

Sheila : Aku tidak yakin (I don't think so). I think we should go straight

Farah: Yah.., Aku tidak yakin. Aku takut kita tersesat.

(Karena Sheila tidak yakin ingin pergi ke kiri.)

B. I don't think so

Because kata "I think we should turn left" yang artinya Si Farah akan belok kanan namu dijawab oleh Si Sheila dengan kata "I think we should just go straight" yang artinya Sheila menginginkan untuk tetap lurus, Jadi Si Sheila pasti menjawab dengan kalimat yang tidak setuju yakni I don't think so/I don't agree with you
