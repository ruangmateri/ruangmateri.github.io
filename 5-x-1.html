25. 5. Tentukan nilai x dari.. │2x - 1│ > │x + 4│a. x < - 1 atau x > 5b. x < 1 atau x > 5c. -1 ≤ x ≤ 5d. -1 ≤ x < 5e. 1 < x ≤ 5​

|2x - 1| > |x + 4|

Gunakan sifat nilai mutlak :

√(2x - 1)² > √(x + 4)²

(2x - 1)² > (x + 4)²

(2x - 1)² - (x + 4)² > 0

Ingat !

a² - b² = (a + b)(a - b)

maka :

[(2x - 1) + (x + 4)][(2x - 1) - (x + 4] > 0

(2x + x - 1 + 4)(2x - x - 1 - 4) > 0

(3x + 3)(x - 5) > 0

• Pembuat nol :

x = -1 atau x = 5

• Garis bilangannya :

+++ (-1) ––– (5) +++

karena bertanda ">", berarti arahnya ke positif.

Jadi, HP = {x | x < -1 atau x > 5; x ∈ R}
