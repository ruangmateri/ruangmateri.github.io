13. sebutkan 3 contoh internet sebagai media komunikasi interaktif


1. VoiP (Voice Over Internet Protocol) : digunakan untuk melakukan pembicaraan telepon melalui jaringan inyernet.

2 IRC (Internet Relay Chat) : merupakan fasiltas untuk berkomunikasi secara interaktif antar pengguna komputer.

3. Internet Phone / CONFERENCE : merupakan fasilitas untuk melakukan percakapan jarak jauh via internet.

4. User's Network (UseNet) adalah sistem kelompok diskusi dimana artikel-artikel didistribusikan ke seluruh dunia.

5. Surfing : aktivitas untuk menjelajah berbagai macam fasilitas di internet.
