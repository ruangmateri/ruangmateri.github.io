22. untuk menggabungkan document inti dengan sumber data supaya diperoleh isi surat yang sebenarnya dipergunakan fasilitas.....A. data sourceB. main documentC. merge the data with the documentD. open the file document​


Pilihan yang paling tepat untuk menggabungkan dokumen inti dengan sumber data pada aplikasi pengolah kata Microsoft Office Word adalah dengan menggunakan fasilitas data source. Berikut ini adalah penjelasan lebih lengkap mengenai data source.

Pembahasan:

Fasilitas atau fitur data source lumrahnya dilakukan untuk aktivitas yang dikenal dengan istilah "Mail Merge" yang pada dasarnya digunakan untuk menggabungkan beberapa dokumen terpisah menjadi sebuah dokumen inti. Misalnya surat bisnis yang didalamnya terkandung teks atau data eksternal yang tersimpan dalam bentuk:

Spreadsheet dari ExcelDaftar kontak di Microsoft OutlookData dari Microsoft Office WordBasis data (database) dari aplikasi Microsoft AccessFile atau data berupa teks polos

Untuk mengakses fitur atau fasilitas data source ini kamu harus memilih tab menu Mailings >> Start Mail Merge >> Step by step Mail Merge Wizzard lalu dilanjutkan dengan memilih sumber data eksternal yang kamu kehendaki. Ilustrasinya bisa dilihat melalui lampiran gambar

Mail merge adalah:

- Merupakan kemampuan komputer untuk mengirimkan kepada beberapa orang sekaligus dari satu sumber surat.
