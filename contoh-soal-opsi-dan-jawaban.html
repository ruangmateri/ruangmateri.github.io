r = (x^2 + 4x + 2)/(x^2 + 6x + 3)

r(x^2 + 6x + 3) = x^2 + 4x + 2

rx^2 + 6rx + 3r = x^2 + 4x + 2

rx^2 + 6rx + 3r - x^2 - 4x - 2 = 0

(r - 1)x^2 + (6r - 4)x + 3r - 2 = 0

a = r - 1

b = 6r - 4

c = 3r - 2

mempunyai akar real yang sama (akar rangkap) , D = 0

b^2 - 4ac = 0

(6r - 4)^2 - 4(r - 1)(3r - 2) = 0

36r^2 - 48r + 16 - 4(3r^2 - 5r + 2) = 0

36r^2 - 48r + 16 - 12r^2 + 20r - 8 = 0

24r^2 - 28r + 8 = 0 (: 4)

6r^2 - 7r + 2 = 0

(3r - 2)(2r - 1) = 0

3r - 2 = 0 atau 2r - 1 = 0

3r = 2 atau 2r = 1

r = 2/3 atau r = 1/2

jawabannya : D

5. Soal Terlampir!!!#NoCopas#PakeCara#Jngn Lngsung Opsi. ​


Nilai dari [tex] \sf \lim\limits_{x \to \pi} \dfrac{sin \: x}{ \sqrt{\pi + tan \: x} - \sqrt{\pi - tan \: x} }[/tex] adalah [tex] \boxed{\sf b. \: - \sqrt{\pi}}[/tex].

Pembahasan:

[tex] \sf \lim\limits_{x \to \pi} \dfrac{sin \: x}{ \sqrt{\pi + tan \: x} - \sqrt{\pi - tan \: x} }[/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x}{ \sqrt{\pi + tan \: x} - \sqrt{\pi - tan \: x} } \times \dfrac{\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x}}{\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x}} [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{( \sqrt{\pi + tan \: x} - \sqrt{\pi - tan \: x})(\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x}) } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{\pi + tan \: x - (\pi - tan \: x) } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{\pi + tan \: x - \pi + tan \: x } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{ tan \: x + tan \: x} [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{ 2 \: tan \: x } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{sin \: x \: (\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x})}{ 2 \: .\dfrac{sin \: x}{cos \: x} } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{(\sqrt{\pi + tan \: x} + \sqrt{\pi - tan \: x}) \: cos \: x}{ 2 } [/tex]

[tex] \sf = \lim\limits_{x \to \pi} \dfrac{\sqrt{\pi + tan \: x} \: . \: cos \: x + \sqrt{\pi - tan \: x} \: . \: cos \: x}{ 2 } [/tex]

[tex] \sf = \dfrac{\sqrt{\pi + tan \: \pi} \: . \: cos \: \pi + \sqrt{\pi - tan \: \pi} \: . \: cos \: \pi}{ 2 } [/tex]

[tex] \sf = \dfrac{\sqrt{\pi + 0} \: . ( - 1) + \sqrt{\pi - 0} \: . (- 1)}{ 2 } [/tex]

[tex] \sf = \dfrac{ - \sqrt{\pi} - \sqrt{\pi }}{ 2 } [/tex]

[tex] \sf = \dfrac{ - 2\sqrt{\pi} }{ 2 } [/tex]

[tex] \sf = - \sqrt{\pi} [/tex]

Jadi, opsi yang tepat adalah (B).

Jawaban serta Pengerjaan :

[tex]{ \boxed{No.1}}[/tex]

Diketahui :

suhu daging = (-3)°Csetelah 30 menit suhu naik = 10°C

Ditanya :

berapa suhu daging sekarang?

Dijawab :

(-3) + 10 = 7

jadi suhu daging sekarang 7°C

[tex]{ \boxed{No.2}}[/tex]

Diketahui :

semut jatuh = 50 cmsemut naik = 40 cmsemut jatuh = 20 cm

Ditanya :

berapa jarak semut dari permukaan lubang?

Dijawab :

50 - 40 + 20 = 30

jadi jarak semut dari permukaan lubang 30 cm

[tex]{ \boxed{No.3}}[/tex]

Diketahui :

suhu gunung saat malam = (-5)°Csuhu gunung saat siang = 15°C

Ditanya :

berapa selisih suhu saat siang dan malam hari di gunung tersebut?

Dijawab :

15 - (-5) = 15 + 5 = 20

jadi selisih suhu saat siang dan malam hari di gunung tersebut 20°C

[tex]{ \boxed{No.4}}[/tex]

Diketahui :

suhu diluar rumah = (-12)°Csuhu didalam rumah = 4°C

Ditanya :

berapa selisih suhu diluar dan didalam rumah?

Dijawab :

4 - (-12) = 4 + 12 ÷ 16

jadi selisih suhu diluar dan didalam rumah 16°C

[tex]{ \boxed{No.5}}[/tex]

Diketahui :

suhu kota dua bulan lalu = (-5)°Csuhu kota sekarang naik = 26°C

Ditanya :

berapa suhu kota tersebut sekarang?

Dijawab :

(-5) + 26 = 21

jadi suhu kota tersebut sekarang 21°C

23. contoh hak opsi dan hak repudiasi


Hak opsi adalah sebuah hak untuk memilih kewarga negaraan: 

Contoh: SI A telah menetap di Amerika selama 20 tahun. ketika ditawari untuk menjadi penduduk USA dengan ID card amerika, jika ia menyetujuinya, maka ia telah menggunakan hak opsi, selama kewarganegaraan RI nya telah dicabut terlebih dahulu

Hak repudiasi adalah hak untuk menolak kewarganegaraan

Contoh: dalam kasus si A, jika ia menolak maka ia menggunakan Hak repudiasi. hak tersebut bisa berlaku jika suatu warga negara telah memiliki kewarganegaraan tertentu yang ada pada paspornya
