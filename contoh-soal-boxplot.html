Jawaban:

a. 99,1

b. k1 = 94.5 and k3 = 102.5

c. k2 = 98

d. IQR atau JAK = 8

e. 95

f. 32.85

g. tidak ada pencilan

Penjelasan:

data = 112, 103, 102, 101, 99, 97, 95, 95, 94, 93

A. RATA RATA

[tex]=\frac{1}{10} (112 + 97 + 102 + 95 + 93 + 95 + 101, 99 + 94 + 103)\\=\frac{1}{10} (991)\\=\frac{991}{10} \\=99.1[/tex]

B. K1 dan K3

93, 94, 95, 95, 97,

K1 = 94+95/2 = 94.5

99, 101, 102, 103, 112

K3 = 102+103/2 = 102.5

C. MEDIAN atau K2

K2 = 112, 103, 102, 101, 99, 97, 95, 95, 94, 93

    = 99 + 97 / 2

    = 196 / 2

    = 98

D. JAK

JAK = K3 - K1

       = 102.5 - 94.5

       = 8

E. MODUS

Modus paling sering muncul,

112, 103, 102, 101, 99, 97, 95, 95, 94, 93

Modus = 95

F. Simpangan Baku

[tex](112-99.1)^{2}=166.41\\(103-99.1)^{2}=15.21\\(102-99.1)^{2}=8,41\\(101-99.1)^{2}=3.61\\(99-99.1)^{2}=0.01\\(97-99.1)^{2}=4,41\\(95-99.1)^{2}=16.81\\(95-99.1)^{2}=16.81\\(94-99.1)^{2}=26.01\\(93-99.1)^{2}=37,21\\\\\\\\=\sqrt{\frac{166.21+15.21+8.41+3.61+0.01+4.41+16.81+16.81+26.01+37.21}{10-1}} \\=\sqrt{\frac{295.69}{9}} \\=32.85[/tex]

Jadi Simpangan bakunya adalah 32.85

G. GAMBAR BOXPLOT, ADA PENCILAN ? JELASKAN !!

Batas atas: K3 + 1.5 x JAK = 102.5 + 1.5 x 8

                                           = 114.5

Batas bawah: K1 - 1.5 x JAK = 94.5 - 1,5 x 8

                                              = 82.5

Karena tidak ada nilai yang berada di luar batas atas dan batas bawah pada rumus boxplot yang telah dihitung sebelumnya. Oleh karena itu, boxplot dapat digambarkan secara normal.

Penjelasan:

Berikut adalah perintah R untuk membuat plot dan sunflowerplot antara variabel Nilai dengan Frekuensi:

```

# Nilai dan Frekuensi

nilai <- c(56, 64, 68, 70, 73, 76, 79, 80, 85, 87)

frekuensi <- c(2, 4, 5, 6, 7, 3, 4, 8, 3, 4)

# Plot

plot(nilai, frekuensi, main = "Plot Nilai vs Frekuensi", xlab = "Nilai", ylab = "Frekuensi")

# Sunflowerplot

library(plotrix)

sunflowerplot(nilai, frekuensi, main = "Sunflowerplot Nilai vs Frekuensi", xlab = "Nilai", ylab = "Frekuensi")

```

Berikut adalah perintah R untuk membuat bentuk Pie dan boxplot pada kedua variabel tersebut:

```

# Pie chart

pie(frekuensi, labels = nilai, main = "Pie Chart Nilai")

# Boxplot

boxplot(nilai, main = "Boxplot Nilai", ylab = "Nilai")

```

Catatan: Perintah R ini diasumsikan Anda telah memasukkan nilai dan frekuensi ke dalam R.

Penjelasan dengan langkah-langkah:

Diketahui:

Data ulangan 30 siswa sebagai berikut:

52, 49, 44, 58, 48, 57, 54, 50, 54, 49, 51, 49, 45, 51, 46, 50, 49, 44, 50, 53, 51, 53, 45, 52, 50, 50, 55, 50, 50, 53.

Ditanya:

Nilai rata-rata, median, modus!

Pembahasan:

Nilai rata-rata

Cara menentukan nilai rata-rata adalah dengan menjumlahkan seluruh data dan membagi dengan jumlah siswa:

[tex]\bar x[/tex] = (52+49+44+58+48+57+54+50+54+49+51+49+45+51+46+50+49+44+50+53+51+53+45+52+50+50+55+50+50+53) : 30

[tex]\bar x[/tex] = 1.512 : 30 = 50.4

Median

Urutkan data mulai dari yang terkecil:

44, 44, 45, 45, 46, 48, 49, 49, 49, 49, 50, 50, 50, 50, 50, 50, 50, 51, 51, 51, 52, 52, 53, 53, 53, 54, 54, 55, 57, 58.

Karena data ada 30, maka nilai tengahnya ada dua yakni data ke-15 dan 16 = 50 dan 50. Sehingga mediannya = (50+50) : 2 = 50.

Modus

Nilai yang sering muncul di data ini adalah 50 yakni sebanyak 7 kali.
