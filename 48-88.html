U18 = a + (18-1)b = 88

= a + 17b = 88

Eliminasi

a + 7b = 48

     -10b = -40

      b = -40

      b = 4

Substitusi

a + 7b = 48

a + 7(4) = 48

a + 28 = 48

a = 48 - 28

a = 20

U21 = 20 + (21-1)4

U21 = 20 + (20x4)
