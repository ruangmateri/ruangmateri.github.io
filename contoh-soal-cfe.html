⇒Jawaban

Besar sudut CFE adalah 70°

⇒Pembahasan

Pertama-tama diketahui sudut DFE sebesar 110°, karena sudut DFE sehadap dengan sudut 110°.

Kemudian sudut EFD berpelurus dengan sudut CFE, sehingga:

∠CFE + ∠DFE = 180°

∠CFE = 180° - ∠DFE

∠CFE = 180° - 110°

∠CFE = 70°0

Jadi besar sudut CFE adalah 70°

19. ^Sebutkan^ nilai x & ∠AEF, ∠BEF, ∠DFE, ∠CFE


Jawab:

∠AEF = 80° (bertolak belakang)

∠AEF + ∠CFE = 180°  (sudut dalam sepihak)

80° + 8x = 180°

8x = 180° - 80°

8x = 100°

x = 12,5°

∠BEF = 180° - ∠AEF  (sudut lurus)

         = 180° - 80°

∠BEF = 100°

∠BEF + ∠DFE = 180°  (sudut dalam sepihak)

100° + ∠DEF = 180°

∠DEF = 180° - 100°

∠DEF = 80°

∠CFE = 8x

∠CFE = 8(12,5°)

∠CFE = 100°
