2. Ada beberapa uji kausalitas, salah satunya model yang dikemukakan oleh Granger. Buatlah model persamaan kausalitas Granger antara ekspor dan GDP!


Jawaban:

Model Persamaan Kausalitas Granger antara Ekspor dan GDP adalah sebagai berikut:

Granger Causality Test:

Model 1: GDP~ Lags(GDP, 1:2)+ Lags(Ekspor, 1:2)

Model 2: GDP~ Lags(GDP, 1:2)

chi2test df p-value

------------------------------------------------------

Model 1 9.582 4 0.048

Model 2 5.729 2 0.057

------------------------------------------------------

Berdasarkan hasil uji kausalitas Granger di atas, dapat disimpulkan bahwa ekspor Granger-menyebabkan GDP jika p-value di bawah 0,05. P-value Model 1 adalah 0,048, sehingga dapat disimpulkan bahwa ekspor Granger-menyebabkan GDP. Sedangkan, P-value Model 2 adalah 0,057, sehingga tidak dapat disimpulkan bahwa GDP Granger-menyebabkan dirinya sendiri.

Dengan demikian, dapat dikatakan bahwa terdapat hubungan kausalitas satu arah dari ekspor terhadap pertumbuhan GDP, yakni ekspor mempengaruhi pertumbuhan GDP.

~ Answer:b. the boys won't build the tent.

.

~ Discussion:

Halo, sobat cerdas! Agar kita dapat merubah pernyataan positif (+) tersebut, kita perlu memerhatikan tense-nya dulu.

» The boys will build the tent.kalimat di atas termasuk kalimat verbal dengan auxiliary: will menunjukkan suatu pekerjaan akan terjadi nanti di masa depan menggunakan bentuk simple futureS + will + V1 + ...

Kalimat negatif (-) menyatakan hal atau pekerjaan yang bertentangan dengan kalimat positif dengan dibuktikan adanya not setelah helping verb.

S + will not + V1 + ...

Will not dapat menggunakan contraction dengan merubahnya menjadi won't.

JAWABANNYA:

b. the boys won't build the tent.

•Run untuk memasukkan sebuah script command.
•Cummads untuk membuka dialog script commands yang akan menampilkan script command yang tersedia dan variable.

•Terminate process untuk menghentikan debug saat ini, dan menutup semua file yang sedang terbuka yang digunakan oleh program.

•Add breakpoint untuk memasukkan sebuah breakpoint.
•Breakpoint Options untuk mengatur breakpoint yang akan dimasukkan.
•Add Watch untuk menetapkan atau sebuah watch exspession.
•Evaluate untuk membuka jendela evaluate.
