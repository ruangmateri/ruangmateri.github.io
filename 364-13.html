28. Suatu persegi panjang mempunyai keliling 80 cm dan luasnya 364 cm², maka bentuk persamaan kuadratnya adalah... a. x² + 40x + 364 = 0 b. x² + 40x – 364 = 0 c. x² – 40x + 364 = 0 d. x² – 40x – 364 = 0 Mohon dengan cara dan penjelasannya kak,,,


Jawaban:

Bangun Ruang

Penjelasan dengan langkah-langkah:

Persegi Panjang

Keliling = 80 cm

Luas = 364 cm²

Keliling = 2 (P+L) = 80

Luas= P x L = 364

2(P+L)=80

(P+L)=80/2

(P+L)=40

(P) = 40-L

(40-L)L=364

40L-L²=364

-L²+40L-364=0

L²-40L+364=0

x²-40x+364=0

⇒Opsi C
