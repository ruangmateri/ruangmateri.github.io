1. Mencari luas sisi alas dan tutup tabung:Diameter = 20 cm, maka r = 20 : 2 = 10 cmLuas alas dan tutup tabung = 2 x π x r x rLuas alas dan tutup tabung = 2 x 3,14 x 10 x 10Luas alas dan tutup tabung = 628 cm2Mencari luas selimut tabung:Luas selimut = panjang x lebar

Panjang = keliling lingkaran

Lebar = tinggi tabung

Keliling lingkaran = π x d

Keliling lingkaran = 3.14 x 20

Keliling lingkaran = 62,8 cm

Luas selimut tabung = keliling lingkaran x tinggi tabung

Luas selimut tabung = 62,8 x 30

Luas selimut tabung = 1.884 cm2

Mencari luas permukaan tabung:

Luas permukaan tabung = Luas alas + tutup + luas selimutLuas permukaan tabung = 628 cm2 + 1884 cm2

Luas permukaan tabung = 2.512 cm2

2. Mencari luas alas dan tutup prisma:

Sisi tegak2 = Sisi miring2 – Sisi alas2

Sisi tegak2 = 202 – 122

Sisi tegak2 = 400 – 144

Sisi tegak2 = 256

Sisi tegak = √256

Sisi tegak = 16 cm

Luas alas + tutup = 2 x (1/2 x alas x tinggi)

Luas alas + tutup = 2 x (1/2 x 24 cm2 x 16 cm2)

Luas alas + tutup = 2 x (1/2 x 384 cm2)

Luas alas + tutup = 2 x 192 cm2

Luas alas + tutup = 384 cm2

Mencari luas sisi tegak prisma:

Luas sisi tegak = panjang x lebar

Panjang = keliling segitiga

Lebar = tinggi prisma

Luas sisi tegak = keliling segitiga x tinggi prisma

Luas sisi tegak = (24 + 20 + 20) x 30

Luas sisi tegak = 64 x 30

Luas sisi tegak = 1.920 cm2

Mencaru luas permukaan prisma:

Luas permukaan prisma = luas tutup + alas + luas sisi tegak

Luas permukaan prisma = 384 cm2 + 1.920 cm2

Luas permukaan prisma = 2.304 cm2
