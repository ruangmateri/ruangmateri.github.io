8. Pengertian network id, host id, dan broadcast id


Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.

28. IPv4 kelas B, 16 bit pertama adalah.... A Network ID B Host id C internet protocol D Router ID E Ip IP addressbantu di jawab mas/mba nya ​


Jawaban singkat:

A Network ID

Penjelasan tambahan:

Keterangan beberapa kelas IP Address versi 4:

Kelas A = Network ID. Host ID. Host ID. Host ID

Kelas B = Network ID. Network ID. Host ID. Host ID

Kelas C = Network ID. Network ID. Network ID. Host ID

Jadi kesimpulannya:

- Network ID dalam kelas A ada di 8 bit pertama

- Network ID dalam kelas B ada di 16 bit pertama

- Network ID dalam kelas C ada di 24 bit pertama
