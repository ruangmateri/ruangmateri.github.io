2. j. Ali, Ben and Betty shared $23 492. Ali got twice as much money as Ben. Ben got twice as much money as Betty. How much money did Ali get?​


Let's use algebra to solve this problem. Let x represent the amount of money that Betty received.

1. Ben got twice as much money as Betty, so Ben received 2x dollars.

2. Ali got twice as much money as Ben, so Ali received 2 * 2x = 4x dollars.

Now, we know that the total amount they shared is $23,492:

x (Betty) + 2x (Ben) + 4x (Ali) = $23,492

Now, add up the money they received:

x + 2x + 4x = 7x

So, 7x = $23,492

Now, divide both sides by 7 to find the value of x (the amount Betty received):

x = $23,492 / 7

x = $3,356

Now that we know the amount Betty received, we can find out how much Ali received:

Ali = 4x

Ali = 4 * $3,356

Ali = $13,424

So, Ali received $13,424.
