19. Alia’s brother wanted to have pen pals too. Alia introduced ______ (he/him/his) to Hannah’s and Saidah’s brothers. Now, they have become good friends. Sometimes Alia writes about _______ (theirs/them) in _______ (hers/her) letter to Hannah and Saidah.


Kelas: SMA
Pelajaran: English
Kategori: Pronoun
Pembahasan:

Alia introduced him.....(the answer should be an object so the right one is 'him' because 'him' is as an object pronoun while 'he' is a subject pronoun and 'his' is a possessive pronoun)
Sometimes Alia writes about them .....(because 'them' is an object pronoun, while 'theirs' is a possessive pronoun)....in her letter ...(because 'her' is a possessive pronoun as the next word is 'letter', an object, so the answer should be possessive pronoun, while 'hers' is also possessive pronoun but without an object aftewards)
