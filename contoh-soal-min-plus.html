7. rumuspenjumlahan dan pengurangan min dan plus


rumus-rumus positif dan negatif

positif = (+)

negatif = (-)


penjumlahan


a) (+) + (+) = (+)


b) (-) + (-) = (-)


c) (+) + (-) = bisa (+) bisa juga (-)

jika (+) lebih besar maka hasilnya (+)

jika (-) lbh besar maka hasilnya (-)


contoh:

9 + (-6) = 3 → hasil positif

9 + (-16) = (-7) → hasil negatif


d) (-) + (+) = bisa (+) bisa juga (-)

jika (+) lebih besar maka hasilnya (+)

jika (-) lebih besar maka hasilnya (-)


contoh:

(-5) + 2 = (-3) → hasilnya negatif

(-5) + 9 = 4 → hasilnya positif


pengurangan


a) (+) - (+) = bisa (+) bisa juga (-)

apabila bilangan pertama lebih besar maka hasilnya positif

apabila bilangan kedua lebih besar maka hasilnya negatif


contoh:

5 - 2 = 3 → hasil positif

5 - 9 = (-4) → hasil negatif


b) (-) - (-) = bisa (+) bisa juga (-)

apabila bilangan pertama lebih besar dari bilangan kedua maka hasilnya (+)

apabila bilangan kedua yang lebih besar maka hasilnya (-)


contoh:

(-2) - (-5) = (-2) + 5 = 3 → hasilnya positif karena -2 (bilangan pertama) lbh besar dari pada -5 (bilangan kedua)

(-6) - (-2) = (-6) + 2 = (-4) → hasilnya negatif karena -6 (bilangan pertama) lebih kecil dari pada -2 (bilangan kedua)


c) (+) - (-) = (+)


d) (-) - (+) = (-)


perkalian

(+) x (+) = (+)

(-) x (-) = (+)

(+) x (-) = (-)

(-) x (+) = (-)


pembagian

(+) : (+) = (+)

(-) : (-) = (+)

(+) : (-) = (-)

(-) : (+) = (-)

30. kenapa min dikurangi min hasilnya plus??


Nah ini pertanyaan nya sedikit ambigu.

Min dikurangi min saya artikan:
-x - (-y)
Sehingga menjadi -x + y
Hal ini dikarenakan min dikurangi min diartikan sebagai tanda min dioperasikan sebagai perkalian.
Dalam hal ini tidak bisa dikatakan hasilnya min. Jika x < y maka hasilnya positof (+)

Namun bila maksudnya:
- x - y
Maka memang benar hasilnya akan min.
Hal di atas saya artikan secara matematis adalah:
- x - y = - (x + y)
Atau
- x - y = - x + (- y)
Yg artinya adalah penjumlahan antara dua buah bilangan negatif pasti akan selalu menghasilkan bilangan negatif yg lebih besar.
