Varian dan Standar Deviasi (Simpangan Baku)

Varian dan standar deviasi (simpangan baku) adalah ukuran-ukuran keragaman (variasi) data statistik yang paling sering digunakan. Standar deviasi (simpangan baku) merupakan akar kuadrat dari varian.

Oleh karena itu, jika salah satu nilai dari kedua ukuran tersebut diketahui maka akan diketahui juga nilai ukuran yang lain.

Penghitungan

Dasar penghitungan varian dan standar deviasi adalah keinginan untuk mengetahui keragaman suatu kelompok data. Salah satu cara untuk mengetahui keragaman dari suatu kelompok data adalah dengan mengurangi setiap nilai data dengan rata-rata kelompok data tersebut, selanjutnya semua hasilnya dijumlahkan.

Namun cara seperti itu tidak bisa digunakan karena hasilnya akan selalu menjadi 0.

Oleh karena itu, solusi agar nilainya tidak menjadi 0 adalah dengan mengkuadratkan setiap pengurangan nilai data dan rata-rata kelompok data tersebut, selanjutnya dilakukan penjumlahan. Hasil penjumlahan kuadrat (sum of squares) tersebut akan selalu bernilai positif.

Nilai varian diperoleh dari pembagian hasil penjumlahan kuadrat (sum of squares) dengan ukuran data (n).

Namun begitu, dalam penerapannya, nilai varian tersebut bias untuk menduga varian populasi. Dengan menggunakan rumus tersebut, nilai varian populasi lebih besar dari varian sampel.

Oleh karena itu, agar tidak bias dalam menduga varian populasi, maka n sebagai pembagi penjumlahan kuadrat (sum of squares) diganti dengan n-1 (derajat bebas) agar nilai varian sampel mendekati varian populasi. Oleh karena itu rumus varian sampel menjadi:

Keterangan:

x = rata-rata

xi = data

n = banyak data

Penyelesaian

Diketahui:

70,68,72,71,69,70

Simpangan baku (S)

x = (70+68+72+71+69+70)/6

x = 420/6

x = 70

2. Masukkan rumus.

S = √(∑(x - xi)²)/n)

= √((70-70)²+(70-68)²+(70-72)²+(70-71)²+(70-69)²+(70-70)²)/6)

= √((0)²+(2)²+(-2)²+(-1)²+(1)²+(0)²)/6)

= √(0+4+4+1+1+0)/6)

= √(10/6)

= √(5/3)

= √5/√3

= √5/√3. √3/√3

= (1/3)√15

Jadi, simpangan baku dari data itu adalah (1/3)√15.

20. simpangan baku dari data tunggal 4,6,7,6,3,4​


Simpangan baku dari data tunggal 4, 6, 7, 6, 3, 4 adalah [tex]\sqrt{2}[/tex]. Simpangan baku adalah salah satu ukuran sebaran dalam statistika. Nama lain dari simpangan baku adalah standar deviasi. Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data. Untuk menentukan simpangan baku, kita harus mencari nilai rata-rata data dan nilai ragam (varians) terlebih dahulu. Simpangan baku dirumuskan sebagai akar kuadrat dari ragam.

Rumus rata-rata

x = [tex]\frac{jumlah \: data}{banyaknya \: data} = \frac{\sum x_{i}}{n}[/tex]

Rumus Ragam (Varians)

R = [tex]\frac{\sum (x_{i} - x)^{2}}{n}[/tex]

Rumus Simpangan Baku

Sb = √R

Pembahasan

Rata-rata dari 4, 6, 7, 6, 3, 4  adalah

x = [tex]\frac{\sum x_{i}}{n}[/tex]

x = [tex]\frac{4 + 6 + 7 + 6 + 3 + 4}{6}[/tex]

x = [tex]\frac{30}{6}[/tex]

x = 5

Nilai dari Ragam (varians)

R = [tex]\frac{\sum (x_{i} - x)^{2}}{n}[/tex]

R = [tex]\frac{(4 - 5)^{2} + (6 - 5)^{2} + (7 - 5)^{2} + (6 - 5)^{2} + (3 - 5)^{2} + (4 - 5)^{2}}{6}[/tex]

R = [tex]\frac{(-1)^{2} + (1)^{2} + (2)^{2} + (1)^{2} + (-2)^{2} + (-1)^{2}}{6}[/tex]

R = [tex]\frac{1 + 1 + 4 + 1 + 4 + 1}{6}[/tex]

R = [tex]\frac{12}{6}[/tex]

R = 2

Nilai dari simpangan baku

Sb = [tex]\sqrt{R}[/tex]

Sb = [tex]\sqrt{2}[/tex]

Jadi simpangan baku dari data tersebut adalah [tex]\sqrt{2}[/tex]

Tentukan simpangan baku dari sekelompok data tunggal 7,3,5,4,6,5!

Jawab:

Ketika sedang belajar matematika maka kita akan belajar tentang statistika, bilangan bulat, bangun ruang, dan lain sebagainya. Simpangan baku adalah suatu bilangan tak negatif yang memiliki satuan yang sama dengan data. Berdasarkan soal, sekelompok data tunggal 7,3,5,4,6,5. Sehingga simpangan baku dari sekelompok data tunggal tersebut yaitu:

Rata-rata = [tex]\frac{7+3+5+4+6+5}{6}=\frac{30}{6}=5.[/tex]

Simpangan baku = [tex]\sqrt{\frac{(7-5)^{2}+(3-5)^{2}+(5-5)^{2}+(4-5)^{2}+(6-5)^{2} +(5-5)^{2} }{6} }[/tex]

Simpangan baku = [tex]=\sqrt{\frac{10}{6} }=\sqrt{\frac{5}{3} } =\frac{1}{3}\sqrt{15}[/tex].

Dengan demikian, simpangan baku dari sekelompok data tunggal 7,3,5,4,6,5 adalah [tex]\frac{1}{3}\sqrt{15}[/tex].

35. Simpangan baku dari data tunggal berikut 7,6,5,5,4,3 adalah​

Penjelasan dengan langkah-langkah:

Untuk mencari simpangan baku dari data tunggal , yakni dengan mencari nilai rata - rata terlebih dahulu ,

3,4,5,5,6,7

-> 3 + 4 + 5 + 5 + 6 +7

-> 30 / (jumlah data )

-> 30 / 6

-> 5 (nilai rata - rata ) kita beri notasi Xi

Setelah itu mencari nilai simpangannnya ,

Σ ( x - Xi)² = (3-5)² + (4-5)² + (5-5 )²+ (5-5)² + (6-5)² + (7-5)²

Σ ( x - Xi)² = 4 + 1 + 0 + 0 + 1 + 4

Σ ( x - Xi)² = 10

Lalu kemudian mencari variannnya caranya ,

S² = [Σ ( x - Xi)² ] / n (jumlah data )

S² = [ 10 ] / 6

S² = 1,666

terakhir nilai varians harus diakar kuadratkan,

S² = 1,666

S = √1,6666

S = 1,288  (Nilai simpangan baku )
