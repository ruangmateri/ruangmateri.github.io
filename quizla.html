32. (7.) quilai maksimum Fungsi F(x) =3 (x+4) ²-6 adalah...​


Jawab:

F(x) = 3 (x+4)^2 - 6

Sumbu Simetri (x)
h = -4Nilai Maksimum (y)
k = -6Titik Puncak
(h,k) = (-4, -6)

ini namanya soal tentang “bentuk lain dari fungsi kuadrat”

F(x) = a (x-h)^2 + k, a ≠ 0

Grafik fungsi tersebut merupakan parabola yang memiliki

Sumbu simetri x = hNilai max atau min y = kTitik puncak (h,k)


Semoga membantu dan semoga bermanfaat :)

Jawab:

-6

Penjelasan dengan langkah-langkah:

[tex]\displaystyle f(x)=y=ax^2+bx+c\\y=a\left [ x^2+\frac{b}{a}~x+\left ( \frac{b}{2a} \right )^2+c-\left ( \frac{b}{2a} \right )^2 \right ]\\y=a\left [ \left ( x+\frac{b}{2a} \right )^2-\frac{b^2-4ac}{4a^2} \right ]\\y=a\left [ x-\left ( -\frac{b}{2a} \right ) \right ]^2+\left ( -\frac{D}{4a} \right )\\y=a(x-h)^2+k[/tex]

h adalah absis titik stasioner (puncak/balik) fungsi kuadrat juga sebagai persamaan sumbu simetri.

k adalah ordinat titik stasioner (puncak/balik) fungsi kuadrat juga sebagai nilai optimum (maksimum/minimum) nya.

Soal tersebut merupakan bentuk vertex f(x) = a(x - h)² + k. Jadi nilai minimum nya -6. Mengapa minimum? Karena h negatif.

Dibuktikan dengan mengubah ke bentuk standar f(x) = ax² + bx + c

f(x) = 3(x + 4)² - 6
